### 1. 7shifts

#### 基本情報

| 項目 | 詳細 |
|------|------|
| 会社名 | 7shifts Inc. |
| 本社所在地 | サスカチュワン州サスカトゥーン、カナダ (トロントにもオフィス) |
| 設立年 | 2014年 (創業者: Jordan Boesch) |
| 資金調達 | 累計 $131.2M (Series C: $80M, 2022年2月, SoftBank Vision Fund 2主導) |
| 従業員規模 | 約303名 (2026年2月時点) |
| 主な展開地域 | 北米 (米国・カナダ中心) |
| 収益 | $72.9M (2024年), 顧客数 650K |

#### 料金体系

| プラン | 月額料金 | 課金単位 | 主な機能 |
|--------|----------|----------|----------|
| **Comp (無料)** | $0 | 1拠点/最大30名 | 基本スケジューリング、タイムクロック、チップ管理、チームチャット |
| **Entree** | $29.99 | /拠点/月 (年払い) | 労働コンプライアンス、スケジュール強制、PTO管理 |
| **The Works** | $49.99 | /拠点/月 (年払い) | チップ・プーリング、売上予測、高度な分析 |
| **Gourmet** | カスタム | /拠点/月 | エンタープライズ機能一式 |

- 初期費用: なし
- 無料トライアル: 14日間
- 契約期間の縛りなし

#### ターゲット市場
- **対応業態**: レストラン専業 (フルサービス、ファストカジュアル、QSR、バー、カフェ)
- **対象規模**: 小規模個人店～大規模チェーン (IHOP, HCI Hospitality等)
- **主要地域**: 北米

#### 機能比較

| 機能 | 対応状況 |
|------|----------|
| シフトスケジューリング | ○ (ドラッグ&ドロップ、テンプレート、AI最適化) |
| タイムクロック (GPS/ジオフェンシング) | ○ (7punches アプリ、半径25m~500m設定可能、写真認証) |
| タイムシート | ○ |
| PTO/有給管理 | ○ (Entreeプラン以上) |
| シフト交換 | ○ |
| チャット/コミュニケーション | ○ (全プランでチームチャット) |
| タスク管理 | ○ (Tipsも含む) |
| 給与連携 | ○ (内蔵ペイロール、ADP、Gusto等) |
| レポート/分析 | ○ (労働コスト、売上予測) |
| モバイルアプリ | ○ (iOS/Android) |
| POS連携 | ○ (Toast, Square, Clover, Lightspeed等 多数) |
| AI予測/最適化 | ○ (売上予測、AIスケジューリング) |

#### 強み (vs SciPOS)
- レストラン業界に完全特化した機能セット (チップ管理、メニューベースの売上予測)
- 無料プランが30名まで利用可能 (寛大)
- 北米のPOS連携が非常に豊富
- エンタープライズ大手との契約実績 (IHOP等)
- On-Demand Pay (即時賃金アクセス) 機能
- 拠点単位課金のため、従業員が多い店舗ではコスト効率が良い

#### 弱み (vs SciPOS)
- 北米市場に偏重、欧州/フィンランドのペイロール連携なし
- Procountor/Netvisor連携なし
- POS一体型ではない (外部POS連携のみ)
- EUR2/user/月と比べると、小規模店舗では7shiftsの方がコスト効率が良いが、大規模店では拠点課金の方が安い可能性
- 2024年に19%のレイオフ実施 (68名削減)

#### 直近1年の動向
- 2025-2026年: IHOP、HCI Hospitalityなどエンタープライズ顧客の拡大
- On-Demand Pay機能をプラットフォームに組み込み
- Deloitte Fast 500リスト入り
- 2025年「Canada's Most Admired Corporate Cultures Award」受賞
- 2025年7月に料金プラン改定
- Par (Brink) POS連携に$10/拠点/月のインフラ費用を追加 (2025年10月)

## SciPOS Shift Managerの競争優位性

### 明確な差別化ポイント

1. **POS一体型** - レストランPOSとシフト管理がネイティブ統合されているのはSciPOSとSling (Toast) のみ。他の全競合は外部POS連携に依存。

2. **EUR2/user/月の価格競争力** - ユーザー単位課金の競合中、最安クラス。同額はSling ($2/user) のみ。Deputyは2.5倍、Plandayは1.5倍高い。

3. **フィンランド/北欧ペイロール連携** - Procountor/Netvisor連携は唯一無二。Plandayはデンマーク+英国+豪州のXero連携のみ。Quinyxは汎用的だがフィンランド特化ではない。

4. **Org-timezone対応** - 組織タイムゾーンベースの表示はほぼ唯一の機能。複数タイムゾーンの組織に対応。

5. **BYOD タブレット型POS** - 専用ハードウェア不要、既存タブレットで運用可能。

### 改善・強化が必要な領域

1. **無料プランの不在** - 7shifts (30名)、Sling (30名)、Homebase (20名)、Shiftbase (75名) が寛大な無料プランを提供。14日間トライアルは業界では短い方 (Planday 30日、Deputy 31日)。

2. **AI/予測機能** - 7shifts、Deputy、Homebase、Quinyxが積極的にAI機能を導入。売上予測ベースの自動スケジューリングは重要な差別化要因に。

3. **外部連携エコシステム** - Deputy (58+連携)、7shifts (多数のPOS連携) と比べ、連携先の数で劣る可能性。

4. **マーケット展開** - 北欧以外の欧州市場 (独・仏・英) への展開戦略が必要。Planday、Tamigo、Skello、Comboが既に主要欧州市場を確保。

5. **ブランド認知** - 7shifts (650K顧客)、Deputy (ユニコーン)、Planday (350K+ユーザー) と比べ、認知度構築が課題。

### 最大の脅威

| 脅威レベル | 競合 | 理由 |
|-----------|------|------|
| **高** | Planday (Xero) | 北欧本拠地、近い価格帯 (EUR2.99)、Xeroの資金力 |
| **高** | Sling (Toast) | POS統合モデルが同一、同価格帯 ($2/user)、上場企業のバック |
| **中** | Tamigo | デンマーク発、レストラン起源、欧州9カ国展開 |
| **中** | Quinyx | スウェーデン発、AI Leader、Strawberry等の北欧大手と契約 |
| **低~中** | 7shifts | 北米中心だが、欧州進出の可能性 |
| **低** | Deputy | 高価格帯、北欧展開なし |
