## 1. Toast

### 基本情報
- **本社**: Boston, MA, USA
- **設立**: 2011年
- **調達/時価総額**: 時価総額 約$17.2B (2026年4月時点、NYSE: TOST)
- **従業員**: 約6,500名 (2025年に800名増員)
- **展開地域**: 米国、カナダ、英国、アイルランド

### 料金体系
- **初期費用**: HW購入費 $0〜(Pay-as-you-goプランの場合)
- **月額**: Starter Kit $0/月、POS Plan $69/月
- **課金単位**: 店舗単位 + 端末追加料金
- **無料プラン/トライアル**: Starter Kit ($0/月、決済手数料割増)
- **HW費用**: $494〜$1,034 (端末構成による)

### 決済手数料
- POS Plan: 2.49% + $0.15 (対面)
- Starter Kit: 3.09% + $0.15 (対面)
- オンライン: 3.50% + $0.15

### ターゲット
- **業態**: FSR、QSR、バー、カフェ、ベーカリー、フードトラック
- **規模**: 小〜中規模 (1〜50店舗)
- **主要地域**: 北米

### 機能

| 機能 | 対応 |
|------|------|
| POS | ○ |
| セルフオーダー | ○ (QRコード、キオスク) |
| KDS | ○ (追加 $25〜$50/月) |
| 在庫管理 | ○ (基本機能内蔵、高度は追加) |
| シフト管理 | ○ (Payroll $69/月+$9/従業員) |
| 予約管理 | △ (外部連携) |
| CRM/ロイヤルティ | ○ (追加 $50〜$75/月) |
| デリバリー連携 | ○ (DoorDash, Uber Eats等) |
| 会計連携 | ○ (QuickBooks, Xero) |
| 多言語 | △ (限定的) |
| オフライン | ○ |
| AI機能 | ○ (AI分析、需要予測) |

### HWモデル
専用Android端末 (Toast Flex, Toast Go, Toast Tap)。自社製造のハードウェアエコシステム。

### 強み
- 飲食店特化型で業界最大のインストールベース (11万店舗超)
- エコシステムが充実 (決済、給与、マーケティング一体型)
- 強力なオンラインオーダリング + デリバリー連携
- Fintech機能 (Toast Capital等の融資サービス)

### 弱み (SciPOS比較)
- 多言語サポートが弱い (アジア言語対応が限定的)
- 決済処理がToast Payment固定 (決済プロバイダ選択不可)
- HWがToast独自仕様のためロックイン
- 海外展開が限定的 (主に北米+英語圏)
- アドオン費用が嵩み実質コストが高い

### 最新動向
- 2025年の年間売上$6.15B達成
- 2025年に800名の従業員を増員
- AI駆動の分析機能を強化中
- 英国・アイルランド市場進出

### SciPOS の差別化ポイント (まとめ)

上記の競合調査を踏まえ、SciPOS が差別化できる主要なギャップ:

1. **多言語対応**: Oracle Simphony、Chowbus以外はアジア言語対応が弱い。SciPOSの多言語機能は大きな差別化要因。

2. **デバイスアグノスティック**: Toast、Clover、TouchBistro等は専用HWまたはiPad限定。プラットフォーム非依存は競争優位。

3. **セルフオーダー統合**: セルフオーダーとPOSの深い統合は、追加課金なしで提供するプレイヤーが少ない。

4. **アジア市場知見**: Chowbus以外にアジア系レストランに本格対応するプレイヤーが少ない。ビュッフェ、セルフオーダーのアジア料理特化機能はユニーク。

5. **オールインワン価格**: 多くのプレイヤーがKDS、ロイヤルティ、オンラインオーダー等を追加課金。Rezku/SkyTab型のオールインクルーシブ戦略が有効。

6. **決済プロバイダ柔軟性**: Toast、Square、Clover、Shift4系はすべて自社決済固定。決済プロバイダの選択自由度は差別化要因。

7. **AI機能**: Toast、Square、Chowbus、PAR等がAI投資を加速。AI機能は今後の必須条件になりつつある。
