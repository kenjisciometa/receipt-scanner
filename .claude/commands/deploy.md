# デプロイ

デプロイを実行してください。

## 手順

1. `git branch --show-current` で現在のブランチを確認する
2. `git status` で未コミットの変更がないか確認する（あれば警告）
3. デプロイ対象の環境をユーザーに確認する（staging / production）
4. プロジェクト内のデプロイ設定（vercel.json, Dockerfile, deploy scripts等）を確認する
5. 該当するデプロイコマンドを実行する
6. デプロイ結果を報告する

## 注意事項

- 本番デプロイは必ずユーザーに確認を取ってから実行する
- 未コミットの変更がある場合は先にコミットを促す
- デプロイ設定が見つからない場合はユーザーに確認する
