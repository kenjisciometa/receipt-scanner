## SciPOS営業の切り口

### 最適な製品

| SciPOS製品 | 適合理由 |
|-----------|---------|
| **BOS（セルフオーダーキオスク）** | **最重要**。パイロット→全店展開の次のステップ。Winposキオスクの代替 |
| **KDS** | Winpos KDSとの統合/リプレース |
| **OSD** | ピックアップ通知。狭い店舗スペースで客の流れを改善 |

### なぜセルフオーダーが必要か

1. **OM自身が未来像を語っている** — カウンター注文の減少を予測
2. **ピーク時の行列** — SC/ガソリンスタンド内の小規模店舗で特に深刻
3. **人件費削減** — フィンランドの高い人件費（飲食業€11-13/時）
4. **若年層の嗜好** — キオスク利用率が高いことがパイロットで実証済み
5. **客単価向上** — セルフオーダーでアップセル+10-20%

### コスト削減ポテンシャル

100店舗×各店舗1名分のピーク時人件費削減 = **年間数百万ユーロ規模の節約可能性**

### 営業ストーリー

> 「Winposのキオスクパイロットの次のステップとして、SciPOSのBOS/KDS/OSD統合ソリューションを提案します。SciPOSはセルフオーダーをネイティブに設計したモダンなプラットフォームで、100店舗以上のFC店舗をクラウドで一元管理できます。」

### vs Winpos（現行POS）

| 項目 | Winpos | SciPOS |
|------|--------|--------|
| コア設計 | 従来型POS（キオスクは後付け） | セルフオーダーをネイティブ設計 |
| コアPOS料金 | 有料（非公開） | **無料** |
| セルフオーダー | パイロット段階 | **本番稼働中** |
| 多言語 | △ | **◎（12+）** |
| BYOD | △ | **◎** |
| OSD | △ | **◎** |

### パイロット提案

- Seinäjoki本拠地の店舗でパイロット実施
- 成功事例をFC本部が他店に推奨
- FCモデルのため、本部の推奨があれば展開がスムーズ
