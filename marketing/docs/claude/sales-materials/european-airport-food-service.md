## 3. テクノロジートレンド

### セルフオーダーキオスク
- 欧州空港の約**58%**が何らかの自動キオスクを導入済み
- 効果: 客単価**10-30%増**、注文処理速度**30%向上**、混雑**最大49%削減**
- **European Accessibility Act（EAA）**が2025-2026年施行 → キオスクのアクセシビリティ基準対応が必須。既存システム入替需要の可能性

### モバイル・QRオーダー
- **Grab（Servy）**: 空港モバイルオーダーの主要プラットフォーム。15空港以上で110以上のレストラン対応
- **Oslo空港**: 24時間セルフサービスフードウォール + QRコード注文
- **Copenhagen空港**: デジタルサイネージ、QRコード栄養表示、「Eat & Fly」クーポン

## 4. 北欧空港のF&B状況

### Helsinki空港（Finavia）

| 項目 | 詳細 |
|---|---|
| **旅客数** | 月間100万人以上 |
| **店舗数** | 約50店舗 + 約50のレストラン・カフェ |
| **最大パートナー** | **SSP Finland**（フードコート運営、Finnairラウンジ含む） |
| **受賞** | 2024年Airport Food Hall of the Year（欧州） |
| **2026年新規** | Gate 30エリアにPure、Picnic、R-kioski等5事業者 |

**Finaviaの戦略**: 「フィンランド体験」を重視。非航空収入の多様化。デジタルタッチポイント最適化が戦略目標。

### Stockholm Arlanda（Swedavia）

- 2024-2025年にMarketplace完成（約11,000m2）
- **HMSHost**: 13レストラン・カフェ、初のフードホール。契約額約8,000万ユーロ/5年
- **SSP**: 約20の新F&Bコンセプト導入

### Copenhagen Kastrup

- **SSP Denmark**が最大（1975年から提携、約35ユニット）
- Eat & Flyウェブサイト、デジタルサイネージ、QRコード活用

### Oslo Gardermoen（Avinor）

- 大規模コンセッション更新: 総額約65億NOK
- 6社が落札: SSP Norway、HMSHost、Bit Union、Reitan、LETT、Joe & the Juice
- **Avolta**: セルフサービスフードウォール + QRコード注文を先行導入

## 5. SciPOS営業の機会

### 参入アングル

| 機会 | 詳細 |
|---|---|
| **SSP Finland経由** | ヘルシンキ空港の最大パートナー。味千ラーメン + 自社ブランド30ユニット |
| **新規オペレーター** | ヘルシンキ空港Gate 30エリアに2026年新規出店する5事業者 |
| **フードホール統合POS** | Helsinki、Stockholm、Osloでフードホール型が主流化。複数テナント統合管理のニーズ |
| **EAA規制対応** | 2025-2026年施行のアクセシビリティ基準。既存キオスクの入替需要 |
| **中小オペレーター** | SSP/Avoltaは自前ソリューション構築するが、Bit Union、LETT等の中小は外部調達 |

### SciPOSの空港向け強み

| 強み | 空港での価値 |
|---|---|
| **12+言語** | 空港では必須（英語+現地語+主要観光客言語） |
| **BYOD** | 専用端末不要で迅速な新規出店に対応 |
| **BOS（セルフオーダーキオスク）** | 空港の高回転ニーズに最適 |
| **KDS** | フードコートでの複数キッチン連携 |
| **OSD** | 注文ステータス表示。空港の待ちエリアで有効 |
| **コア無料** | 中小オペレーターの参入障壁を下げる |

### 営業ターゲット優先順位

| 優先度 | ターゲット | 理由 |
|---|---|---|
| **最高** | **SSP Finland** | ヘルシンキ空港最大パートナー。味千含む30ユニット |
| **高** | **ヘルシンキ空港の新規オペレーター** | 2026年Gate 30エリアの5事業者。新規出店時のPOS選定 |
| **高** | **Finavia直接** | デジタルタッチポイント最適化が戦略目標。F&Bパートナー向けソリューション提案 |
| **中** | **HMSHost（Arlanda）** | 新Marketplace。フードホール統合POS需要 |
| **中** | **Oslo空港の中小オペレーター** | Bit Union、LETT等。コンセッション更新でPOS選定中 |
