# フィンランド国内 日本食レストラン市場

**作成日**: 2026年5月4日
**目的**: フィンランド国内の日本食レストランの全体像把握と追加ターゲットの特定

---

## 1. 市場概要

- フィンランドの全レストランの約**6%**が日本食レストラン
- エスニック飲食卸売市場: 2023年3.83億ユーロ → 2028年4.68億ユーロ予測
- 欧州日本食レストラン市場: 2024年37.7億USD、CAGR 3.17%

## 3. Momotoko倒産（2025年5月）

| 項目 | 詳細 |
|---|---|
| **運営会社** | Cathywong Oy |
| **最盛期** | 7店舗（ヘルシンキ、エスポー、ヴァンター、タンペレ、トゥルク） |
| **倒産時期** | 2025年5月 |
| **創業者** | Menkki Kam & Haina Shi（中国系） |
| **原因** | 2024-2025年のフィンランド経済低迷（倒産件数1996年以来最悪）+ 経営要因 |

**影響**: ラーメンセグメントに市場の空白が発生。Fat Ramen等の残存プレーヤーにチャンス。新規参入者がPOSを必要とする可能性。

---

## 4. セグメント別の競争状況

| セグメント | 飽和度 | コメント |
|---|---|---|
| **寿司チェーン** | 高 | Hanko Aasia（22店）が圧倒的。参入障壁高い |
| **ラーメン** | **低**（空白あり） | Momotoko倒産で7店分の空白。Fat Ramen（3店）が主要。**チャンス大** |
| **高級日本食** | 低 | Shii、Gaijin等の少数精鋭 |
| **フュージョン・アジアン** | 中 | Hanko Aasiaのリブランドが示す成長トレンド |

## 5. SciPOS営業ターゲット優先順位

| 優先度 | ターゲット | 理由 |
|---|---|---|
| **高** | **Fat Ramen**（3店舗） | 独立系・成長中・拡大計画あり |
| **中** | **Sushi Yama フィンランド**（4店舗） | フィンランド展開初期。投資ファンド支援 |
| **中** | **Hanko Aasia / NoHo Partners**（22店舗） | 規模最大だがNoHo本社経由の提案が必要 |
| **中** | **Momotoko跡地の新規参入者** | 市場の空白を狙う新チェーンを監視 |
| 低 | 独立系高級店 | スケールメリット限定的 |
