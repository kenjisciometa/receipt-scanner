## 1. Restolution — フィンランド最大の競合（★★★★★）

| 指標 | データ |
|------|--------|
| 顧客数 | 4,000+店舗（FI/DK/NO） |
| 年間売上 | €14M |
| 創業 | 1984年（40年超） |
| PE投資 | 2023年12月 Valedo Partners |
| 料金 | **完全非公開** |
| 主要顧客 | Pizza Hut FI、Löyly、Friends & Brgrs |

### 弱点
1. **1984年創業のレガシー技術** — モダンなクラウドネイティブではない可能性
2. **料金完全不透明** — 比較検討不可。PE投資後の値上げリスク
3. **シフト管理・AI機能なし** — 外部ツール依存
4. **CRMが外部依存**（Cluby連携）
5. **BYOD非対応** — 専用HWで初期コスト高
6. **5言語のみ** — SciPOSの12+に対して限定的
7. **インベントリで39%ユーザーエラー** — 使いにくさの証拠

### スイッチングトリガー
- Valedo PE投資後の値上げ発生時
- シフト管理統合の要望
- 多言語対応ニーズ（外国人スタッフ増加）
- モダンUI/UXへの要望

### バトルカードトーク
> 「Restolutionの40年の実績は、同時に40年の技術的負債を意味します。PEファンドが入った今、値上げは避けられません。SciPOSは2020年代設計のモダンアーキテクチャで、コア無料・BYOD・12言語・シフト管理統合を実現しています。」

## 2. Tebi — 成功報酬型のオランダ新鋭（★★★★☆）

| 指標 | データ |
|------|--------|
| 調達額 | €56M（CapitalG/Index Ventures） |
| 価格 | 月商€10K毎に€20/月。€10K以下は**無料** |
| 決済 | Interchange++方式 |
| 展開国 | NL/UK/ES/IT/CZ/**Fィンランド** |
| 創業者 | Adyen共同創業者系 |

### 弱点
1. **KDSがクラウド依存** — 「常時同期、ローカルサーバー不要」= ネット断線でKDS停止リスク。SciPOSのWebSocketローカル同期と対照的
2. **フィンランド会計ソフト連携なし** — Netvisor/Procountor非対応
3. **セルフオーダーが限定的** — Self Station相当の割り勘・ビュッフェモードなし
4. **シフト管理なし**
5. **フィンランドはまだ初期段階** — 専用サポート体制不明

### スイッチングトリガー
- フィンランド会計連携（Netvisor/Procountor）が必要になった時
- KDSがネット障害で停止した経験がある時
- 割り勘・ビュッフェ対応のセルフオーダーが必要な時

### バトルカードトーク
> 「Tebiは決済出身でPOSに進出したプレイヤーです。決済の透明性は優れていますが、レストラン運営の深い機能（KDSオフライン、シフト管理、フィンランド会計連携）はカバーしていません。」

## 3. Flipdish — ユニコーンだが信頼性に問題（★★★☆☆）

| 指標 | データ |
|------|--------|
| 調達額 | $157M（Tencent主導） |
| 評価額 | $1.25B（ユニコーン） |
| 顧客数 | 7,500+店舗、32カ国 |
| 料金 | POS+注文バンドル €99-129/月 |
| AI | AI Phone Agent（2025年11月ローンチ） |

### 弱点（実際のレビューベース）
1. **ピーク時クラッシュ** — 「注文がキッチンと同期しない」「システムクラッシュ」のTrustpilot/Capterra報告
2. **導入後サポート低下** — 「対応が遅い」「矛盾する情報」「未解決チケット」
3. **顧客データ所有権の制限** — Joint Controller Model
4. **北欧プレゼンスが薄い** — 主力はアイルランド・英国
5. **アジア言語非対応**
6. **予約機能なし**

### バトルカードトーク
> 「Flipdishはユニコーンですが、ユーザーレビューではピーク時のクラッシュや注文同期の問題が報告されています。SciPOSはWebSocketベースの同期でこうした問題を構造的に防いでいます。」

## SciPOSの全競合共通の訴求ポイント

1. **コアPOS無料** — 全競合が有料（Tebiの月商€10K以下無料を除く）
2. **BYOD** — 専用HW不要で初期投資ゼロ
3. **12+言語** — 多国籍スタッフ・顧客に対応
4. **KDSオフライン耐性** — WebSocket+ローカルサーバーでネット断線時も稼働
5. **Self Station** — 割り勘・ビュッフェモード・セッションコード（競合にほぼない）
6. **シフト管理統合** — GPS打刻・タイムシート・有給管理（POS競合の大半が非搭載）
7. **フィンランド会計連携** — Netvisor/Procountor/Fennoa（TebiもMunchiもallOも未対応）
