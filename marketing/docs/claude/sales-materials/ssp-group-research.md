## 2. テクノロジー戦略

### デジタル注文: 全トランザクションの**31%**がデジタル経由（FY2025）

| 施策 | 詳細 |
|------|------|
| **QRコード注文** | テーブルQR→メニュー→注文→決済→チップまで完結。アプリ不要 |
| **セルフオーダーキオスク** | タッチスクリーン注文→カード/デジタルウォレット決済→キッチン自動送信 |
| **チェックアウトフリー** | Amazon Go型の決済レス技術を実験中 |
| **効果** | 客単価10-30%増、サービス速度向上、注文精度向上 |

### POS標準

**グローバル統一POSプラットフォームは公表されていない。** Oracle MICROS、NCR Aloha等との公式パートナーシップ情報もなし。→ **各地域で異なるPOSを使用している可能性が高く、統一プラットフォーム提案の余地あり。**

### 意思決定構造

- 組織は地域別（Europe、Americas、APAC/MEA）
- **各地域のMDが大きな裁量を持つ**
- SSP Finlandのような小規模市場では**ローカルMDが直接の意思決定者**

## 4. SciPOS営業の切り口

### 最大の機会: SSP Finlandの30以上のユニット全体

味千ラーメン1店舗だけでなく、**SSP Finlandが運営する30以上のユニット全体**にSciPOSを提案できる可能性がある。

| ブランド | SciPOS製品 | 理由 |
|---|---|---|
| **味千ラーメン（200席）** | BOS + KDS + OSD | 空港の高回転ニーズ。多言語必須 |
| **Tori Kitchen & Cafe** | BOS + KDS | SSP自社ブランド。POS選定の自由度が高い |
| **Pala Pizza Bar** | BOS + KDS | 同上 |
| **Asian Corner** | BOS + KDS | 同上 |
| **Fafa's（空港店）** | BOS + KDS | 空港店はSSP運営。本体とは別系統 |

### アプローチ戦略

1. **Elena Heiska（MD）に直接アプローチ** — 124名の小組織。意思決定が速い
2. **味千ラーメンを入口に、自社ブランド店舗への展開を提案** — 味千は外部フランチャイズだが、Tori Kitchen等の自社ブランドはPOS選定の自由度が高い
3. **「デジタル注文比率31%を更に上げる」というSSPの全社目標に合致** — セルフオーダーキオスク/QRオーダーの提案が歓迎される環境
4. **欧州鉄道事業のレビュー中** — 変革期にあり、新規テクノロジー導入に前向きな時期
