# SciPOS 製品サマリー（営業用）

**作成日**: 2026年5月4日
**目的**: 営業先への説明・資料用の製品概要

---

## コンセプト

**SciPOS** は、レストラン・飲食店向けの**オールインワン型クラウドPOSシステム**。
BYOD（Bring Your Own Device）モデルで、既存のタブレットにアプリをインストールするだけで導入可能。高額なPOS専用端末は不要。

---

## 製品ポートフォリオ

### 無料コアプロダクト（4製品）

| 製品 | 説明 | 価格 |
|------|------|------|
| **POS** | 注文管理、決済処理、在庫追跡、売上分析、テーブルマップ、マルチ店舗管理 | **無料** |
| **KDS** | キッチンディスプレイ。リアルタイム注文表示、3段階ステータス管理 | **無料** |
| **CDS** | カスタマーディスプレイ。注文内容・金額のリアルタイム表示 | **無料** |
| **SDS** | サーバーディスプレイ。注文完了通知、配膳指示 | **無料** |

### プレミアムプロダクト

| 製品 | 説明 | 価格 |
|------|------|------|
| **BOS（Bell Order System）** | セルフオーダーキオスク。タッチスクリーン注文、ビジュアルメニュー、複数決済、QRレシート | **€90/月（店舗単位）** |
| **TOS（Table Order System）** | テーブルオーダー。QRコードでセッション開始、テーブルからの直接注文、追加注文対応 | **€90/月（店舗単位）** |
| **Self Station** | セルフチェックアウト。テーブルマップ、セッションコード、割り勘（均等/アイテム別）、ビュッフェモード | **€90/月（店舗単位）** |
| **OSD（Order Status Display）** | 注文状況表示。「調理中」「準備完了」表示、音声通知 | **€90/月（店舗単位）** |
| **Shift Manager** | シフト管理。スケジューリング、GPSジオフェンス打刻、タイムシート、有給管理、チャット | **€2/月（従業員単位）** |

### その他

| 製品 | 説明 |
|------|------|

## 競争優位性

| 優位性 | 詳細 |
|---|---|
| **コアシステム無料** | POS + KDS + CDS + SDSが無料。競合は月額€50-150 |
| **BYOD** | 既存タブレットで動作。専用端末€2,000-5,000が不要 |
| **オールインワン** | POS〜セルフオーダー〜キッチン〜シフト管理まで一元管理 |
| **多言語** | 12+言語。メニュー・UI・レシートすべて多言語対応 |
| **オフライン耐性** | LAN経由でPOS↔KDS連携維持。競合（Tebi等）はクラウドオンリーでKDS断絶リスクあり |
| **ビュッフェモード** | 食べ放題/ビュッフェ対応。競合にはほぼない機能 |
| **NFC/QR/ESL** | NFCタップ注文、QRコード注文、電子棚札連携 |
