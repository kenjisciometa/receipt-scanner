## 2. 最優先ターゲット: Rolls（100+店舗）

| 項目 | 詳細 |
|------|------|
| **店舗数** | 100店舗以上（フィンランド全国） |
| **業態** | ハンバーガーファストフード |
| **立地** | ショッピングセンター・サービスステーション中心 |
| **運営** | Jokes Family Oy（フランチャイズ、加盟金・違約金なし） |
| **セルフオーダー** | **導入情報なし** |

**SciPOS提案ポイント**:
- 100+店舗でセルフオーダー未導入は大きな機会
- SC立地はセルフオーダーキオスク（BOS）との相性が最高
- フランチャイズモデルのため、個別店舗からの試験導入が可能
- ファストフード業態はシンプルなメニュー構成でBOS導入が容易

## 3. 高優先ターゲット: Social Burgerjoint

| 項目 | 詳細 |
|------|------|
| **店舗数** | 約9店舗 |
| **業態** | プレミアムバーガー |
| **新オーナー** | Artisan Fast Food Oy（2026年3月にKotipizza Groupから買収） |
| **POS** | 移行期。旧Kotipizzaグループのシステムから変更の可能性 |

**SciPOS提案ポイント**:
- 新オーナーへの移行期 = **POS刷新の最適なタイミング**
- 店舗数は少ないが成長ポテンシャルあり
- プレミアムバーガー業態はTOS（テーブルオーダー）との相性も良い

## 5. フィンランドPOS市場の競合状況

| POS | 主要顧客 | 強み |
|---|---|---|
| **Restolution** | 4,000+店舗。Friends & Brgrs等 | フィンランド市場シェア最大 |
| **Winpos** | Espresso House等 | フィンランド製 |
| **Yonoton** | Subway Finland等 | オムニチャネル（POS+キオスク+アプリ） |
| **Solteq** | 中〜大規模チェーン | ERPとの統合 |
| **Oracle MICROS** | Fafa's等 | グローバル大手 |

**SciPOSの差別化**: コア無料、BYOD、ビュッフェモード、NFC注文、Self Station（割り勘）、12+言語。既存POSが弱いセルフオーダー・テーブルオーダー領域での補完提案が有効。
