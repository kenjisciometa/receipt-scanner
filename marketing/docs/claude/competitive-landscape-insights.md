## 2. 各カテゴリの競合状況

### New Ventures（61社）
ロボティクス、音声AI、フードウェイスト、ゴーストキッチン等の「未来型」企業群。大半がPOSを持たず補完関係。diniiのみ日本で直接競合。

### POS（90社）
最も競争が激しい。地域・規模・業態で棲み分け:
- 北米: Toast（120,000店）、SpotOn（$3.6B評価）
- 欧州: Lightspeed、iiko/Syrve、各国ローカルPOS
- 日本: スマレジ（ARR 100億円）、Airレジ（無料）、dinii（3,000店）
- 中東: FOODICS、Petpooja

### Self-Ordering（86社）
キオスク市場は$28B超・CAGR 15-20%で急成長中。最も直接的な競合カテゴリ。

### Shift Management（22社）
成熟市場。AI需要予測が唯一の差別化要素。POS統合型は少数派。

### Restaurant Marketing（21社）
ローカルSEO・レビュー管理・SNS管理に特化。POS連携が弱い企業が多い。

### Restaurant SaaS（76社）
予約管理、CRM、デリバリー統合、決済等の専門SaaS。LINEヤフー/トレタ連合の登場が日本市場の最大の変数。

## 3. SciometaPOSへの戦略的示唆

### 現状の差別化ポイント

1. **POS + セルフオーダー + シフト管理の垂直統合** — フルスタック提供はToast・Flipdish程度で少数
2. **多言語・多地域対応** — 12言語以上は業界で希少
3. **BYOD + Android対応** — 専用HW強制モデル（Toast等）に対する低コスト優位性
4. **キオスク型セルフオーダー** — LINEレストランプラスやdiniiがQRモバイルオーダー中心なのに対し差別化
5. **アジア市場フォーカス** — 北米・欧州大手の未参入領域

### 未開拓の機会

| 機会 | 根拠 | 難易度 |
|------|------|--------|
| AI需要予測 → 自動シフト生成 | POS売上データ自社保有の強み。統合提供は極めて少ない | 中 |
| CRM/ロイヤルティ機能 | セルフオーダーデータは宝の山。SevenRooms=$1.2B買収が価値を証明 | 中 |
| Embedded Finance（融資） | POS売上データ基盤の融資は新たな収益源 | 高 |
| フードウェイスト削減 | POS/在庫データ活用。規制強化の追い風 | 低 |
| GEO（AI検索最適化） | 市場初期段階。構造化データの提供で先行者利益 | 低 |
| ダイナミックプライシング | POS+セルフオーダー統合だからこそ実現しやすい | 中 |

### 警戒すべき脅威

1. **LINEレストランプラス（最大の脅威）** — 2026年6月開始、初年度無料、LINE 1億人ユーザー。ただしキオスク型は弱い
2. **dinii** — 74.6億円調達、3,000店舗。極めて類似のポジショニング
3. **Toast/SpotOnの国際展開** — Toast英国進出済み。アジア進出を注視
4. **CloudKitchens Otter** — 米国デリバリー注文18%処理。$19/月の低価格POS
5. **Noryのアジア展開** — AIバックオフィスとして急成長中

## 4. 面白い・意外な発見

### パラドックスと逆説

- **Miso Roboticsの「PoC地獄」**: $125M調達→14台導入。「すごい技術」≠「スケーラブルなビジネス」の最も鮮明な実例
- **iiko → Syrve の分裂**: 地政学リスク（ロシア-ウクライナ）がSaaS企業のブランド・事業を分断。国際展開時のカントリーリスクの重要性を示す

### 驚異的な資本効率

- **Skello**: Series B $54.5Mのみで ARR €52.5M達成。追加調達なしの成長は驚異的
- **GoTab**: ブルワリー・フードホールという超ニッチ特化で150%成長。「全レストラン」を狙うより特定ニッチの深耕が有効

### 日本市場固有の動き

- **LINEヤフーの本気度**: トレタ買収→LINEレストランプラス→初年度無料は、日本レストランテック業界の勢力図を根本的に書き換える可能性
- **diniiの「推しエール」**: スタッフへのチップ機能をモバイルオーダーに実装。日本の飲食文化に新しい収益モデルを提案
- **グローリーのShowcase Gig買収**: 旧来のキャッシュレジスター企業がセルフオーダーに参入。異業種からの脅威

### 新しいカテゴリの創出

- **顔認識ロイヤルティ（GRUBBRR）**: キオスクで顔認識リピーター自動識別。GDPRとの緊張関係があるが、アジア市場では受容可能性
- **GEO（AI検索最適化）**: Google SEOの次のフロンティア。ChatGPTに「渋谷のラーメン屋」と聞かれたとき上位に出る仕組み
- **Fleksa（独）**: POS+マーケティング統合を14名で推進。SciometaPOSと最も類似した統合モデルだが極めて小規模 — この「空白地帯」はチャンス
