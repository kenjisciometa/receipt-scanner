## 1. 過去5年の決算推移

良品計画（東証7453）は2020年に決算期を2月末→8月末に変更。2020年8月期は6ヶ月の変則決算。

| 決算期 | 営業収益 | 営業利益 | 純利益 | 備考 |
|--------|---------|---------|--------|------|
| 2020年2月期 | 4,387億円 | 363億円 | 232億円 | 減益基調 |
| 2020年8月期 | 1,793億円 | 8.7億円 | **▲169億円** | 6ヶ月変則決算・コロナ直撃 |
| 2021年8月期 | 4,876億円 | 約450億円 | 約339億円 | V字回復・過去最高益水準 |
| 2022年8月期 | 4,961億円 | 327億円 | 245億円 | 増収減益（原材料高・円安） |
| 2023年8月期 | 5,814億円 | 331億円 | 220億円 | 増収・利益横ばい |
| 2024年8月期 | 6,616億円 | 561億円 | 415億円 | 営業利益+69%、過去最高益 |
| 2025年8月期 | 7,846億円 | 738億円 | 508億円 | 全利益段階で過去最高更新 |

### 主要な転換点

1. **2020年 — コロナ危機**: 減損損失142億円計上、最終赤字169億円。**米国子会社がChapter 11申請**。海外不採算店の閉鎖・撤退
2. **2021年 — V字回復と社長交代**: 堂前宣夫氏が社長就任（ファーストリテイリング出身）。「第二創業」宣言。**2030年目標: 売上3兆円、営業利益率15%**
3. **2022-2023年 — 踊り場**: 原材料費高騰・円安・中国景気減速で利益低迷。国内価格改定を段階的に実施
4. **2024-2025年 — 再成長**: 価格改定効果とコスト管理が奏功。3期連続過去最高益を見込む

### 欧米セグメント業績

- FY25/8通期: 営業収益421億円（+7.9%）、セグメント利益69億円（+25.7%）
- FY26/8 Q2: 営業収益248億円（+17.1%）、2桁成長達成
- ただし全社売上の**約5.4%**にとどまり、アジアとの規模差は大きい

### 2024年の経営破綻と再編

- **Muji Europe Holdings Limited（MEH）**：良品計画66%・三菱商事34%の合弁（2007年設立）
- 2024年4月にAdministration（英国式経営破綻手続き）に入る
- 原因: COVID期の€2,640万の株主借入が返済不能に
- **プリパック取引**で良品計画の完全子会社「MUJI Europe Limited」に事業譲渡
- 三菱商事との合弁を解消し、**完全直営体制へ移行**。730名の雇用維持

### 欧州の課題（アジアとの比較）

| 課題 | 詳細 |
|---|---|
| 収益構造の脆弱性 | 2024年に経営破綻。アジアは同時期に増収増益 |
| 店舗規模の不足 | 欧州32店 vs 東アジア400店超。スケールメリットが出にくい |
| 高コスト構造 | 欧州の人件費・賃料が高い |
| ブランド認知度 | アジアに比べ浸透に時間がかかる |

### 2025-2030年 欧州旗艦店出店計画

| 年 | 都市 | 詳細 |

### フィンランド — MUJI Kamppi Helsinki

- **2019年11月8日オープン**。フィンランド初かつ唯一のMUJI店舗
- **場所**: ヘルシンキ中心部、Kamppiショッピングセンター4階全フロア
- **売場面積**: 約3,600m2 — オープン時点で**欧州最大のMUJI店舗**
- **特徴**:

  - **MUJI Local Design Village** — フィンランドのデザイナー・食品メーカーのショップインショップ
  - **MUJI Small Gallery** — 地元アーティスト展示スペース
- フィンランドのデザイン文化との親和性を活かした**北欧向けモデル店舗**として位置付け

## 4. MUJIの飲食事業

### 全体規模 — 世界で約40〜43拠点

|---|---|---|---|
| **Cafe&Meal MUJI / Cafe MUJI** | 26 | 7〜9 | 33〜35 |

### 展開ブランド詳細

#### Cafe&Meal MUJI（最大ブランド）

- 「素の食」コンセプト。約20種類のデリから選べるスタイル
- **国内26店舗**: 東京（新宿、渋谷西武、有明、上野マルイ、吉祥寺マルイ等）、神奈川（鎌倉、青葉台、新百合丘、聖マリアンナ医科大学病院）、大阪（グランフロント、堺北花田）、京都（BAL、山科）、兵庫（神戸BAL）等
- **海外7〜9店舗**: 香港4店（皇室堡、美麗華廣場、奧海城、又一城）、シンガポール2〜3店（Plaza Singapura、Raffles City等）、台湾1店、中国本土0〜1店
- 「Cafe MUJI」（ドリンク・軽食中心の小型カフェ）4店舗も含む
- **海外は全てアジアに集中。ヨーロッパにはCafe&Meal MUJIは存在しない**
- 2025年1月に聖マリアンナ医科大学病院内に出店（良品計画初の病院内店舗）

#### MUJI Diner（フルレストラン）

- 「素の食」をコンセプトとした本格的食事。世界各地の食文化・未利用魚等を活用
- **銀座**（MUJI銀座 B1F）— 朝食7:30〜、ランチ、ディナー22:00まで
- **深圳**（MUJI HOTEL SHENZHEN 2F）— 6:30〜23:00
- **北京**（MUJI HOTEL BEIJING内）— 天安門広場を望むレストラン

#### MUJI Ravintola（ヘルシンキ）

- Kamppi Shopping Centre 4F（MUJI店舗と同フロア）
- **ヨーロッパ初かつ唯一のMUJIレストラン**
- 「一汁三菜」ベース、フィンランド地元食材で日本食を再現
- メニュー例: 味噌ラーメン（€15.50）、照り焼きサーモン、タパス、サンドパン
- ヴィーガン・ベジタリアン対応、テイクアウト・予約可
- ランチ（平日11:00〜15:00）、スイーツ・デリは毎日、ディナー（木〜土）
- **MUJI Ravintola Dining** — より親密なディナー空間を同施設内に併設
- **MUJI Local Food Store併設** — フィンランド100社以上の地場生産者の商品
- Tripadvisor評価: 4.7/5.0（ヘルシンキ2,021軒中152位）
- MUJI直営と見られる

#### MUJI Hotel内レストラン

- **MUJI HOTEL GINZA**: WA（和）レストラン（6F、郷土料理を3ヶ月ローテーション）+ ベーカリー（1F、国産小麦）+ Salon
- **MUJI HOTEL SHENZHEN / BEIJING**: MUJI Diner（上記）

#### MUJI Food Market（新業態）

- **2025年1月ニューヨークにオープン**。MUJIとして米国初のフードマーケット
- Chelsea Market内。おにぎり、味噌汁、カレー、どら焼き等
- AIコーヒーロボット「Jarvis」搭載
- 物販ではなく飲食提供型の新業態

### 飲食事業の拡大見通し

- フードサービス単独の拡大計画の公式発表はないが、**フラッグシップ出店に伴う飲食拠点の増加**が見込まれる
- 2026年秋冬のパリ旗艦店（欧州最大級）にレストランが併設される可能性あり
- 2025年11月バンコク Central World に東南アジア最大店舗（フード関連含む可能性）
- ヘルシンキのRavintolaが好評 → 欧州追加展開のモデルケースになりうる

### ヨーロッパでの飲食事業の位置づけ（営業上の重要ポイント）

- **MUJI Ravintola ヘルシンキはヨーロッパ唯一のMUJI飲食拠点**
- 海外のCafe&Meal MUJIは全てアジア（香港・シンガポール・台湾）に集中
- ヘルシンキでの飲食オペレーションの成功・改善が、今後の欧州展開のモデルに直結する
- パリ旗艦店にレストランが入る場合、ヘルシンキでの知見が参照される可能性が高い

### 現在の飲食POS・オーダーシステム

- **poscube（iPad POSレジ）**: Cafe&Meal MUJIの一部店舗で導入
- iPad miniによるスタッフオーダー方式、バーコード伝票連携
- **顧客セルフオーダーは飲食では未導入**

## 5. テクノロジー基盤・デジタル戦略

### 物販POS・セルフサービス

| 技術 | 詳細 |
|---|---|
| **ID-POS** | AWS Data Lake + Kinesis → Athena → Tableau Server |
| **NCRセルフレジ** | 2015年導入（食品販売以外で日本初） |
| **MishiPay** | 英国全店 + スウェーデンでScan & Go + セルフチェックアウトキオスク導入。**平均客単価25%増**。フィンランド等への展開は未確認 |

### MishiPay（欧州物販セルフチェックアウト）— 重要

- 2020年秋ロンドン店で導入 → 全英店舗に展開。2021年2月スウェーデンにも導入。フィンランド含むその他欧州への展開は未確認
- スマホで商品スキャン→決済（アプリDL不要）
- 導入フロアは有人レジ2台→MishiPay完全移行
- **スタッフを接客に再配置**する成功体験あり

### デジタルエコシステム

| システム | 詳細 |
|---|---|
| **MUJI Passport** | ロイヤリティアプリ。累計7,853万DL。購買データ統合 |
| **EC（欧州）** | 2024年5月にBigCommerceへ移行。Adyen/PayPal決済 |
| **Treasure Data** | オンライン・オフライン行動データ統合（2014年〜） |
| **prismatix** | API基盤、統一顧客体験 |
| **Helpfeel** | AI-FAQ（2024年導入、問い合わせ70%削減） |

- 国内・海外ともに年60店純増

- **「感じ良いくらし」**: テクノロジーは「はしゃぎ過ぎず」取り入れる

- **個店経営**: 各店舗の自律性を重視
- **コオウンド経営**: 顧客・地域との共同所有意識
- **ESG経営**: サステナビリティをトップランナーとして推進

## 7. パートナーシップの切り口

### 最有力ポイント

1. **飲食セルフオーダリングの空白**: 物販ではMishiPayで先進的セルフサービスを実現済み。しかし**飲食ではスタッフオーダーに留まり、顧客セルフオーダーは未導入**
2. **ヘルシンキが最適な実証拠点**: 欧州初・最大のMUJI飲食拠点。フィンランド企業Sciometa Oyとの距離的・文化的親和性
3. **MUJIの「感じ良い」哲学との整合**: ミニマルなUXのセルフオーダーシステムはMUJIブランドと高い親和性
4. **MishiPayの成功体験**: 物販で「スタッフを接客に再配置」した効果を飲食でも再現できる論理
5. **OMO戦略への貢献**: MUJI Passport連携、飲食購買データ統合によるロイヤリティ強化の可能性
6. **欧州事業再建中**: コスト効率とオペレーション改善に敏感な時期

### SciPOS製品とMUJIニーズの対応

| MUJIの状況 | SciPOS製品 | 提案価値 |
|---|---|---|
| 飲食セルフオーダー未導入 | **Table Order System (TOS)** | テーブルサイドQR注文 |
| スタッフオーダーの効率化 | **Bell Order System (BOS)** | カウンターセルフオーダー |
| 会計の効率化 | **Self Station** | セルフチェックアウト・分割払い |
| キッチン連携 | **Kitchen Display System (KDS)** | リアルタイム注文追跡 |
| 注文ステータス通知 | **Order Status Display (OSD)** | 顧客向けステータス表示 |

## 8. 営業アプローチの留意点

### Do（推奨）

- **「共創・実験」のトーンで**: 売り込みではなく「一緒に試してみませんか」
- **フィンランド発を強調**: 地域土着化を重視するMUJIに対し、北欧の価値観（シンプル・サステナブル）を共通言語にする
- **MishiPayの成功に言及**: 「物販で実証された効果を飲食でも」という論理展開
- **段階的アプローチ提案**: まずMUJI Ravintola Helsinkiでの小規模PoC → 成果次第で展開
- **既存システムとの共存**: poscube等既存POSとの連携可能性を示す
- **多言語対応を訴求**: 12+言語対応はMUJIの国際展開と合致

### Don't（避けるべき）

- **ハードセルは厳禁**: MUJIの意思決定は慎重・合議的。相手は営業を煙たがっている
- **コスト削減だけを前面に出さない**: MUJIは「感じ良い体験」が最優先。効率化は手段
- **テクノロジーの押し売り**: 「最先端」より「自然な体験」を語る
- **初回で契約を急がない**: 今回は関係構築が目的

### 会話の切り口案

1. MUJI Ravintolaの体験を褒める（実際に食事してから行くと良い）
2. フィンランドの飲食業界のデジタル化トレンドを共有
3. MishiPayの物販での成果に触れ、飲食での可能性を示唆
4. SciPOSのデモを見せる（押し付けず「こんなものを作っています」程度）
5. 「もしご興味あれば、Ravintolaで試験的に使ってみていただくことも可能です」
