# Accountant Portal - Marketing Page Spec

## Product Overview
The Accountant Portal allows accountants to view restaurant sales and shift attendance data in real-time. The accountant app lives at https://tax.sciometa.com/.

## User Flow
1. Accountant creates an account at tax.sciometa.com
2. An Account ID is generated and shared with the restaurant owner
3. Owner enters the Account ID in POS Settings → External Access
4. Accountant gets real-time access to sales + shift attendance data

### Features (6 cards)
1. **Real-time Sales Dashboard** - Live sales data from connected restaurants
2. **Shift & Attendance Data** - Employee hours, clock-in/out records
3. **Multi-Restaurant Support** - Manage multiple restaurant clients from one dashboard
4. **Custom Reports** - Generate financial reports filtered by date, location, category
5. **Secure Data Sharing** - Read-only access via unique Account ID, no POS credentials needed
6. **Automated Summaries** - Daily/weekly/monthly financial summaries delivered automatically

### Benefits (6 items)
1. Real-time financial visibility without visiting the restaurant
2. Reduce manual data entry and transcription errors
3. Support multiple restaurant clients from one dashboard
4. Secure, read-only access to financial data
5. Automated report generation saves hours of manual work
6. Seamless integration with existing POS workflow

### How It Works (4 steps)
1. **Create Account** - Register at tax.sciometa.com
2. **Get Account ID** - Share your unique ID with restaurant owners
3. **Owner Connects** - Restaurant owner enters Account ID in POS Settings → External Access
4. **Access Data** - View real-time sales and attendance data from your dashboard

### Roles (3 cards)
1. **Accountant** - View sales data, attendance records, generate reports
2. **Restaurant Owner** - Connect accountant via Account ID, control data access
3. **Admin** - Manage multiple connections, configure data sharing policies
