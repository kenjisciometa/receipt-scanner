## 5. 主要プロダクト・機能一覧

### プロダクト体系

Restolabesはレストラン向けゼロコミッション・オンライン注文プラットフォーム。オンライン注文を中心に、ブランドアプリ・QRオーダー・デリバリー管理を統合。

### 機能一覧

| 機能カテゴリ | 対応状況 | 詳細 |
|-------------|---------|------|
| **POS レジ** | △ | 自社POSなし。Toast/Clover/Revel/Square/PAR等と連携（Checkmate経由含む） |
| **オンライン注文** | ◎ | コア機能。ピックアップ・デリバリー・ケータリング対応 |
| **QRコードオーダー** | ○ | QRテーブルオーダー（Premiumプラン以上） |
| **セルフオーダーキオスク** | × | キオスク機能なし |
| **KDS** | △ | POS連携経由でKDS対応。自社KDSなし |
| **在庫管理** | △ | メニューストックカウンター（Premiumプラン以上）。基本的な在庫管理 |
| **シフト・勤怠** | × | 非対応 |
| **予約** | × | 非対応（テーブル予約機能なし） |
| **CRM・ロイヤルティ** | ○ | ロイヤルティスイート内蔵。ポイント・リワード・クーポンビルダー。Como連携（Enterprise） |
| **デリバリー連携** | ◎ | DoorDash Drive、Uber Direct、Tookan、Postmates/UberEats、Shipday |
| **会計連携** | × | 直接的な会計ソフト連携情報なし |
| **レポート** | ○ | リアルタイムアナリティクス。売上・注文・顧客データ |
| **多言語** | × | 英語中心。多言語対応の明示なし |
| **オフライン** | × | クラウドベース。オフライン対応なし |

| **モバイルアプリ** | ○ | ブランドiOS/Androidアプリ（Proプラン or $49/月アドオン） |

| 強み | 詳細 |

| **ゼロコミッション** | 取引手数料なし。レストランが100%利益保持 |

| **デリバリー連携充実** | DoorDash/Uber Direct等との直接連携 |
| **決済ゲートウェイ豊富** | 50+ゲートウェイ対応。地域に応じた最安処理料率の選択が可能 |
| **30日無料トライアル** | クレカ不要で全機能試用可 |
| **ブランドアプリ** | iOS/Androidネイティブアプリ（追加料金） |

## 11. 弱み（SciometaPOS比較視点）

| 弱み | 詳細 |
|------|------|
| **POS非搭載** | 自社POSなし。外部POS必須で追加コスト発生 |
| **キオスク非対応** | セルフオーダーキオスク機能なし |
| **KDS非搭載** | 自社KDSなし。POS経由の間接対応のみ |
| **シフト管理なし** | スタッフ管理・勤怠機能が完全欠如 |
| **予約機能なし** | テーブル予約が非対応 |
| **在庫管理が基本的** | ストックカウンターのみ。本格的な在庫管理なし |
| **多言語非対応** | 英語中心。アジア市場・多国籍対応に弱い |
| **オフライン非対応** | クラウド専用。ネットワーク障害時にリスク |
| **小規模チーム** | 24名。大規模開発・サポート能力に限界 |
| **AI機能なし** | AI・自動化機能の実装なし |

**SciometaPOS優位点**: SciometaPOSはPOS・セルフオーダー・KDS・シフト管理・在庫管理を統合した単一プラットフォーム。Restolabsはオンライン注文特化でPOSや店内オペレーション機能が欠如しているため、フルスタック運営にはSciometaPOSが大幅に優位。多言語対応（日本語・アジア言語）も差別化要素。
