## 14. Prestop（プレストップ）

### 基本情報
| 項目 | 詳細 |
|------|------|
| 正式社名 | Prestop B.V. |
| 本社 | Veghel, Netherlands |
| 設立 | 2010年 |
| 代表 | Johan Vendrig |
| 従業員数 | ~30名 |
| URL | prestop.com |

### 展開市場
- **主力**: オランダ、ベルギー、DACH
- **展開中**: 欧州全域
- **主要顧客**: Febo（オランダ最大ファストフード）、Julia's、HEMA、Albert Heijn

### 料金体系
- ハードウェア販売型（キオスク端末 + 設置サービス）
- ソフトウェアは要問い合わせ
- カスタム端末製造にも対応

### 主要機能
- **セルフオーダーキオスク製造**（15"/22"/27"/32"の各サイズ）
- **壁掛け型/スタンド型/カウンター型**の豊富なフォームファクター
- **屋外キオスク**（防水・耐候性）
- **決済端末統合**（Adyen, CCV等）
- **ソフトウェアパートナー連携**（自社ソフトなし、サードパーティSW利用）

### POS連携
- 自社ソフトなし。POS/注文ソフトウェアパートナーと連携（Untill, Ready2Order等）

### SciPOS比較・脅威度: **低**
**Prestopの特徴:**
- キオスクハードウェア製造に特化（ソフトウェアは自社開発なし）
- 30名規模の中小メーカー

**SciPOSの優位点:**
- ソフトウェアプラットフォーム（Prestopはハードウェアのみ）
- フルスタック一体型

**結論:** 直接競合ではなく、むしろ**潜在的パートナー**。SciPOSのソフトウェアをPrestopのキオスクハードウェアで動かす連携が考えられる。
