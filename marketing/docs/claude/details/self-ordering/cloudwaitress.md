## 5. 主要プロダクト・機能一覧

### プロダクト体系

CloudWaitressはコミッション無料のオンライン注文プラットフォーム。テイクアウト・デリバリー・QRオーダー・テーブル予約を統合。

### 機能一覧

| 機能カテゴリ | 対応状況 | 詳細 |
|-------------|---------|------|
| **POS レジ** | △ | 簡易ビルトインPOS + 外部POS連携（Litespeed、Abacus POS等1,200+連携） |
| **オンライン注文** | ◎ | コア機能。テイクアウト・デリバリー・スケジュール注文 |
| **QRコードオーダー** | ◎ | QRコード + NFC両対応。テーブル番号自動認識 |
| **NFC注文** | ◎ | NFCタグタップでメニュー表示・注文。市場でも稀少な機能 |
| **セルフオーダーキオスク** | × | キオスク機能なし |
| **KDS** | × | 専用KDSなし。プリンター経由で注文出力 |
| **在庫管理** | △ | メニュー品切れ管理・ホリデー計画機能。本格在庫管理ではない |
| **シフト・勤怠** | × | 非対応 |
| **予約** | ○ | テーブル予約機能あり（Standardプランに含む） |
| **CRM・ロイヤルティ** | △ | プロモーションコード対応。本格的なロイヤルティプログラムは限定的 |
| **デリバリー連携** | ○ | Uber Direct連携。セルフデリバリー管理。ゾーン別配送料・最低注文額設定 |
| **会計連携** | × | 直接的な会計ソフト連携情報なし |
| **レポート** | ○ | パフォーマンスレポート（全プラン） |
| **多言語** | △ | 多通貨対応あり。多言語メニューの明示は限定的 |
| **オフライン** | × | クラウドベース。オフライン対応なし |
| **AI・自動化** | △ | 2025年にAIパーソナライゼーション導入。ピークタイム自動待ち時間調整 |
| **ウェブサイトビルダー** | ○ | カスタマイズ可能なランディングページ・注文ウェブサイト。カスタムドメイン+無料SSL |
| **モバイルアプリ** | ○ | iOS/Androidアプリ（ワンタイム構築費$399 + 月額サブスク） |

## 9. 強み（SciometaPOS比較視点）

| 強み | 詳細 |
|------|------|
| **業界最安級** | $39/月で無制限注文。無料プランも提供 |
| **コミッション完全無料** | 取引手数料ゼロ |
| **QR + NFC両対応** | NFCタグ注文は市場でも稀少。SciometaPOSのNFC対応と類似 |
| **テーブル番号自動認識** | QR/NFCスキャン時にテーブル自動判定 |
| **2012年〜の実績** | 10年以上の運営実績 |
| **Uber Direct連携** | 自社配達なしでもデリバリー対応可能 |
| **無料プラン** | 100件/月まで無料。初期投資ゼロで開始可 |
| **AIパーソナライゼーション** | 2025年にAI機能追加 |

## 10. 弱み（SciometaPOS比較視点）

| 弱み | 詳細 |
|------|------|
| **POS機能が簡易** | ビルトインPOSはあるが本格POSには程遠い |
| **キオスク非対応** | セルフオーダーキオスクなし |
| **KDS非搭載** | キッチンディスプレイなし。プリンター依存 |
| **シフト管理なし** | スタッフ管理・勤怠機能なし |
| **在庫管理が基本的** | 品切れ管理のみ。本格在庫管理なし |
| **CRM/ロイヤルティが弱い** | プロモコードのみ。ポイント・リワードプログラムなし |
| **極少人数チーム** | 6〜7名。開発速度・サポート能力に制約 |
| **多言語対応が限定的** | アジア言語対応の明示なし |
| **オフライン非対応** | クラウド専用 |

**SciometaPOS優位点**: SciometaPOSはPOS・キオスク・KDS・シフト管理・在庫管理の統合プラットフォーム。CloudWaitressはオンライン注文+QR/NFC注文に特化しており、店内オペレーション全般はカバーしない。NFC対応で一部競合するが、フルスタック統合ではSciometaPOSが大幅に優位。
