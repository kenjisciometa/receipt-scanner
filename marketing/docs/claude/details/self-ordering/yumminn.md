### 破産の経緯と教訓
- 3年間全期間赤字、売上€25万以下（マイクロ企業）
- 手数料モデル（0.9%/取引）がスペインの銀行手数料（0.36%）の2.5倍で競争力なし
- TheForkが同等機能を**無料**で提供開始 → 致命的打撃
- 消費者の決済習慣を変えることに失敗
- 350-500店舗導入もユーザー定着せず

### SciPOSへの示唆（脅威度: **低** = 消滅済み）
1. **QR決済単体では持続不可能** → フルスタックの一部として統合するSciPOSのアプローチは正しい
2. **取引手数料モデルに注意** → 既存銀行手数料や無料競合と比較される
3. **消費者習慣の変革は困難** → テクノロジーの押し付けではなく、店舗運営の効率化として訴求すべき
4. Google口コミ連動・ロイヤルティ機能は参考にする価値あり
