## 12. 脅威度: 低

### 理由

**DOYOが優位な点:**
1. **22言語対応**: 業界トップクラスの多言語メニュー自動翻訳。SciPOSの12+言語を上回る
2. **完全ハードウェアフリー**: スマートフォンのみで全機能が利用可能。初期投資ゼロ
3. **低価格**: $30/月からの料金設定で導入障壁が極めて低い
4. **欧州フィスカルコンプライアンス**: 12カ国で税制対応済み
5. **迅速オンボーディング**: 24時間以内に稼働可能

**SciPOSが優位な点:**
1. **フルスタック統合**: POS + 専用キオスクステーション + KDS + シフト管理 + 在庫 = DOYOを大きく上回る統合度
2. **セルフオーダーステーション**: 専用ハードウェアを含む本格的なキオスクソリューション。DOYOのキオスク機能は基本的
3. **NFC タップ注文**: DOYO未対応
4. **ビュッフェモード**: DOYO未対応（業界でもほぼ唯一）
5. **シフト管理**: GPS打刻・タイムシート・PTO等のフル機能。DOYOは基本的なスケジューリングのみ
6. **顧客基盤**: SciPOSは実稼働顧客が拡大中。DOYOは7カ国12レストランと極めて初期段階
7. **無料コアPOS**: SciPOSの無料プランはDOYOの$30/月と比較して導入障壁がさらに低い

**脅威が低い理由:**
1. **極めて初期段階のスタートアップ**: 7カ国12レストランという顧客基盤は商用化初期段階。市場への影響力はほぼゼロ
2. **未調達**: $1.5Mの資金調達を2024年から模索中だが完了情報なし。成長資金の欠如
3. **小規模チーム**: 推定10名未満のチームで、プロダクト開発・営業・サポートの全てをカバーする必要がある
4. **キオスク機能の深度不足**: DOYOのキオスクは基本的なモジュールであり、SciPOSの専用セルフオーダーステーションとは機能・完成度に大きな差がある
5. **北欧・フィンランド市場への進出兆候なし**: SciPOSの主要市場とは重ならない

**注意点:**
- DOYOの22言語自動翻訳機能は、SciPOSが多言語対応を拡充する際の参考になる
- ハードウェアフリーモデルは、小規模店舗や新興市場へのアプローチとして有効な戦略
- フィスカルコンプライアンスへの注力は、欧州展開を考える際に重要な視点
- 現時点では脅威ではないが、万が一資金調達に成功した場合は再評価が必要
