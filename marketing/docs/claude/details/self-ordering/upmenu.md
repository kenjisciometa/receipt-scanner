## 12. 脅威度: 低

### 理由

**UpMenuが優位な点:**
1. **マーケティングスイート**: メール/SMS/プッシュ通知のオートメーション、ロイヤルティプログラムが充実
2. **自社ブランド化**: レストラン独自のウェブサイト・モバイルアプリ構築が可能
3. **コミッション無料**: 月額固定でフードデリバリーアプリからの脱却を訴求
4. **40+カ国展開**: グローバルな導入実績
5. **デリバリー連携**: Glovo/Wolt/Uber Eats等との注文統合

**SciPOSが優位な点:**
1. **フルスタック統合**: POS + キオスク + KDS + シフト管理 + 在庫 = UpMenuが提供できない統合度
2. **セルフオーダーキオスク**: UpMenu未対応。SciPOSの主要差別化要素
3. **NFC タップ注文**: UpMenu未対応
4. **ビュッフェモード**: UpMenu未対応（業界でもほぼ唯一）
5. **シフト管理**: GPS打刻・タイムシート・PTO等のフル機能
6. **多言語対応**: 12+言語（UpMenuは具体的な言語数不明）
7. **無料コアPOS**: SciPOSの無料プランはUpMenuの$49/月と比較して導入障壁が極めて低い

**脅威が低い理由:**
1. **異なる製品カテゴリ**: UpMenuはオンライン注文・マーケティングプラットフォームが中心。SciPOSはフルスタックPOS + セルフオーダーキオスク。直接競合する領域が限定的
2. **資金力の限界**: ブートストラップで26名のチーム。急速な市場拡大は困難
3. **キオスク非対応**: セルフオーダーキオスク/ステーション領域ではゼロ競合
4. **POS機能の深度**: UpMenuのPOSは2024年に本格開始したばかりで、機能の成熟度はSciPOSに及ばない
5. **北欧市場での存在感なし**: フィンランド・北欧市場への進出兆候は確認されず

**注意点:**
- UpMenuのマーケティングオートメーション・ロイヤルティ機能は、SciPOSが将来的に強化すべき領域として参考になる
- コミッション無料モデルの訴求は、フードデリバリー手数料に不満を持つレストラン層へのアプローチとして有効
