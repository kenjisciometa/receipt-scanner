## 12. 脅威度: 中

### 理由

**Storyousが優位な点:**
1. **5,500+ SMBの顧客基盤**: 中東欧で大きな市場シェアを持つ。実績と信頼がある
2. **Teya/SaltPayの資金力**: 親会社が$845Mを調達済み。成長投資の原資が潤沢
3. **決済統合の深さ**: Teya決済ターミナルとの完全統合は、ハードウェア+ソフトウェアの一体ソリューションとして強力
4. **10年以上の実績**: 2012年創業。チェコ市場では「ラブブランド」。長期の信頼関係
5. **金融サービス展開**: POS→決済→融資→銀行口座のフルスタック金融エコシステム。これはPOS単体企業が真似できない
6. **6カ国での多国展開**: CZ/PL/ES/SK/HR/HUで稼働中。欧州内での展開実績

**SciPOSが優位な点:**
1. **フルスタック統合**: POS + キオスク + KDS + シフト管理 + 在庫 = Storyousが提供できない統合度
2. **セルフオーダーキオスク**: Storyous未対応。SciPOSの主要差別化要素
3. **KDS**: Storyous未対応
4. **NFC タップ注文**: Storyous未対応
5. **ビュッフェモード**: Storyous未対応（業界でもほぼ唯一）
6. **シフト管理**: GPS打刻・タイムシート・PTO等のフル機能。Storyousは未対応
7. **多言語メニュー**: SciPOSの12+言語メニューはStoryousを上回る
8. **無料コアPOS**: SciPOSの無料プランはStoryousの非公開価格と比較して導入障壁が低い

**脅威が「中」である理由:**
1. **異なる地理的市場**: Storyousは中東欧+スペイン中心。SciPOSの主要市場（北欧・フィンランド）とは現時点で重ならない。ただし、Teya傘下での欧州拡大が進めば重複リスクあり
2. **強力な親会社**: Teya（$845M調達）が後ろ盾。資金・決済インフラ・人材の全てにアクセス可能。「スタートアップ vs 大企業」の構図になりうる
3. **決済エコシステムの差別化**: POS + 決済 + 融資 + 銀行口座の統合は、SciPOSが短期的に追随できない領域
4. **ただし機能面の差**: KDS・キオスク・シフト管理・NFC・ビュッフェといったSciPOSのコア差別化機能がStoryousにはない。飲食店オペレーションの深度ではSciPOSが優位
5. **地域限定性**: 5,500+の顧客の大半はチェコ国内。北欧・西欧への本格進出はまだ限定的

**注意点:**
- Teya傘下での欧州拡大動向を継続的にモニタリングすべき。特にTeyaが北欧の決済市場に参入する場合、Storyousも連動して展開する可能性がある
- 決済手数料ベースのビジネスモデルは、POS自体を無料/低価格で提供する余裕がある。価格競争では不利になる可能性
- Storyousの「POS + 決済 + 金融」統合モデルは、中長期的なSciPOSの戦略策定の参考になる
- Borgun（アイスランド）買収は北欧圏への関心を示唆。今後の動向に注意
