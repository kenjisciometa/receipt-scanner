## 強み（SciometaPOS比較視点）

1. **マーケティング統合POS**: POS + キオスク + ロイヤルティ + マーケティング自動化を1プラットフォームで提供。「マーケティング中心のPOS」という独自ポジション
2. **ゲーミフィケーション**: キオスクにインタラクティブなゲーム機能を搭載。顧客エンゲージメントの差別化
3. **柔軟な契約**: 長期契約不要、キオスク返却自由。導入障壁が低い
4. **DoorDash手数料無料**: デリバリー連携でコミッション無料を実現
5. **AI電話注文**: AI搭載の電話注文自動化
6. **デジタルビルボード**: キオスクアイドル画面をプロモーション用デジタルサイネージとして活用
7. **ピーク時90%キオスク処理**: 労働コスト40%削減、注文サイズ15%増加を実績として謳う

## 弱み（SciometaPOS比較視点）

1. **極小チーム**: 従業員17名は非常に小規模。サポート・開発リソースに不安
2. **北米限定**: 海外展開の情報なし
3. **在庫管理不明**: 在庫管理機能の有無が不明確
4. **シフト・勤怠管理なし**: SciPOSのようなシフト管理機能の記載なし
5. **調達額は多いが詳細不明**: $14M調達の内訳が不明確（Seedが$1.37MならSeries Aで$12.6M?）
6. **レビュー・実績が少ない**: Capterra/G2等のレビューサイトでの情報が乏しい
7. **他社POS非連携**: 独自POS前提。既存POS利用店舗には移行コスト

## 調査メモ

- Peblla は小規模だが「マーケティング中心POS + キオスク」というユニークなポジション
- バブルティー/カフェ市場でのニッチ確立を狙う戦略
- $14M調達の内訳が不明確（PitchBookとCrunchbaseで情報が異なる可能性）
- 2025-2026年の公開情報が少なく、事業継続状況の確認が必要
- SciPOSは規模・機能の幅広さで優位だが、ゲーミフィケーションや契約柔軟性は参考にすべき
