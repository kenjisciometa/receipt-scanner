## 2. honei（ホネイ）

### 基本情報
| 項目 | 詳細 |
|------|------|
| 正式社名 | YOPIDOYA, S.L. |

| 従業員数 | 推定20-35名 |

### 展開市場
- **主力**: スペイン（バルセロナ・マドリード中心）
- **展開中**: ポルトガル
- **計画中**: 中南米（LATAM）
- **規模**: 300+ブランド導入、月間トランザクション€1M超、前年比売上5倍成長

### 料金体系
- **初期費用・月額**: なし（ゼロコスト導入）
- **収益モデル**: トランザクションベースのコミッション型（具体率は非公開、推定1.5-3%）
- Honei Terminal（専用決済端末）による追加収益

### 主要機能
- QRオーダー＆ペイメント（POS連携でキッチン直接送信）
- Honei Terminal（インテリジェント決済端末、2024年10月ローンチ）
- **Bee Loyal**（ロイヤルティ、カード決済から自動顧客識別 → 60%会員化率実績）
- データ分析・AIマーケティング
- 割り勘機能
- Google口コミ2倍増効果

### POS連携
Sinqro経由で **30以上のPOS** と連携（Revo XEF、Techni-Web等スペイン系POS中心）

### SciPOS比較・脅威度: **低〜中**
**honeiの優位点:**
- ゼロコスト導入モデル（導入障壁が極めて低い）
- Bee Loyalのデータ駆動CRM/ロイヤルティ
- スペイン飲食文化への深い理解

**SciPOSの優位点:**
- フルスタックPOS + セルフオーダー + シフト管理
- キオスク端末あり（honeiはQRのみ）
- 多国籍展開対応

**注意点:** スペイン/LATAM市場への進出時に間接的競合となる可能性。Bee LoyalのCRM機能は参考にすべき。
