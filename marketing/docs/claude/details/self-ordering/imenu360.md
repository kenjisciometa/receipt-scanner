## 5. 主要プロダクト・機能一覧

### プロダクト体系

iMenu360はフランチャイズ・チェーン向けホワイトラベルオンライン注文プラットフォーム。リセラー・エンタープライズ向けの大規模展開に対応。

### 機能一覧

| 機能カテゴリ | 対応状況 | 詳細 |
|-------------|---------|------|
| **POS レジ** | △ | 自社POSなし。18+POS連携（Micros、Toast、Clover、Lavu、Lightspeed、PAR、Bepoz、Breadcrumb等） |
| **オンライン注文** | ◎ | コア機能。ブランドウェブサイト・モバイルアプリ経由の直接注文 |
| **QRコードオーダー** | ○ | QRコードコンタクトレスオーダー対応 |
| **セルフオーダーキオスク** | ○ | キオスク対応（get.imenu360.comで明示） |
| **KDS** | × | 専用KDS機能の明示なし |
| **在庫管理** | × | 在庫管理機能の明示なし |
| **シフト・勤怠** | × | 非対応 |
| **予約** | × | 非対応 |
| **CRM・ロイヤルティ** | ○ | ロイヤルティプログラム、クーポンエンジン、初回注文割引、最低購入額で無料アイテム |
| **デリバリー連携** | ○ | ラストマイルデリバリー（自動/手動割当）。Uber Eats/DoorDash/Grubhub連携。Deliverect連携 |
| **会計連携** | × | 直接的な会計ソフト連携情報なし |
| **レポート** | ○ | リアルタイムダッシュボード、アナリティクス |
| **多言語** | × | 英語中心。多言語対応の明示なし |
| **オフライン** | × | クラウドベース |
| **AI・自動化** | × | AI機能の明示なし |
| **モバイルアプリ** | ○ | ブランドiOS/Androidアプリ |
| **ウェブサイト** | ○ | ブランド注文ウェブサイト |
| **ホワイトラベル** | ◎ | リセラー向けホワイトラベルプラットフォーム。マルチテナント管理 |

## 6. 料金体系

| 項目 | 内容 |
|------|------|
| 初期費用 | **$99**（ワンタイムセットアップ） |

| コミッション | **3%**（売上の3%、上限$150/月） |

| プラン | 内容 |
|--------|------|
| 標準プラン | $99初期 + $20/月 + 3%（上限$150/月） |

## 10. 強み（SciometaPOS比較視点）

| 強み | 詳細 |
|------|------|
| **17年+の実績** | 2007年創業。業界最古参級の安定性と信頼性 |
| **フランチャイズ特化** | 集中管理、マルチロケーション対応。チェーン展開に最適化 |
| **コミッション上限制** | 3%だが月額$150上限。高売上店舗には実質定額 |
| **ホワイトラベル** | リセラー向けプラットフォーム。パートナーエコシステム構築 |
| **POS連携豊富** | 18+POS連携。既存POS環境への導入障壁が低い |
| **米軍施設展開** | ニッチだが安定した大口顧客基盤 |
| **10,000+同時注文** | エンタープライズグレードのスケーラビリティ |
| **キオスク対応** | セルフオーダーキオスクも提供 |

## 11. 弱み（SciometaPOS比較視点）

| 弱み | 詳細 |
|------|------|
| **POS非搭載** | 自社POSなし。外部POS必須 |
| **KDS非搭載** | キッチンディスプレイなし |
| **在庫管理なし** | 在庫管理機能なし |
| **シフト管理なし** | スタッフ管理・勤怠機能なし |
| **予約機能なし** | テーブル予約が非対応 |
| **多言語非対応** | 英語中心。アジア市場対応に弱い |
| **コミッションあり** | 3%コミッションは上限ありだが、ゼロコミッション競合に劣る |
| **UI/UXの古さ** | 2007年創業で長期運営だが、UIの近代化が遅れている印象 |
| **AI機能なし** | AI・自動化機能なし |
| **CRM/ロイヤルティが基本的** | クーポン・基本ロイヤルティのみ。高度なCRMは欠如 |
| **チーム規模不透明** | 従業員数に矛盾情報。サポート体制が不透明 |

**SciometaPOS優位点**: SciometaPOSはPOS・キオスク・KDS・シフト管理・在庫管理の統合プラットフォーム。iMenu360はオンライン注文+フランチャイズ管理に特化しており、店内オペレーション機能が薄い。フランチャイズ管理能力は高いが、フルスタック統合ではSciometaPOSが優位。
