## 強み（SciometaPOS比較視点）

1. **AI音声技術のリーダー**: Presto Voiceはドライブスルー音声AI市場でのパイオニア。会話型AI注文自動化の実績が豊富
2. **大規模チェーンの実績**: Top 20レストランチェーンの過半数と取引。300,000+システム出荷、月間1億人以上利用
3. **MIT起源の技術力**: 創業者のMITバックグラウンドに加え、Sam Altman（OpenAI CEO）がエンジェル投資家
4. **マルチユースHW**: Presto Flexはキオスク・テーブルトップ・ハンドヘルド・ドライブスルーと1台で複数用途に対応
5. **ドライブスルー特化**: QSRドライブスルー市場での独占的ポジション

## 弱み（SciometaPOS比較視点）

1. **ドライブスルー偏重**: Presto Phoenixはドライブスルー音声AIに注力しており、キオスク・テーブルオーダーは二次的
2. **オールインワンではない**: POS、在庫管理、シフト管理、デリバリー連携等のフルスタック機能がない
3. **財務不安定の歴史**: 上場後に経営難→資産売却→非公開化という経緯。再建中
4. **エンタープライズ専用**: 個人店・中小店舗向けのプランがなく、大規模チェーン専用
5. **料金不透明**: 全ての料金が非公開
6. **QRオーダー・オンラインオーダー不明**: SciPOSのような多チャネル注文体験の情報がない

## 最近の動向（直近1年）

- **資産売却 → Presto Phoenix**（2025年1月）: Remus Capital主導のコンソーシアムへの資産売却完了。$18Mの新規資本注入。Presto Phoenixとして再スタート
- **$10M追加調達**（2026年1月）: Metropolitan Partners Group主導。Remus Capital, Link Ventures, 戦略的エンジェル投資家が参加。AIドライブスルー事業加速
- **Voice AI特化**: 非公開化後はPresto Voice（ドライブスルーAI音声注文）に事業を集中
- **顧客拡大**: Taco John's, Hardee's/Carl's Jr., Fazoli's等のQSRチェーンでVoice AI展開
- **347名体制**: 6大陸にまたがるチーム維持

## 調査メモ

- Presto PhoenixはPrestoの事業承継会社。旧Presto Automation（PRST）は2021年上場→2024年資産売却
- 現在の主力製品はPresto Voice（ドライブスルーAI音声）であり、キオスクやテーブルトップは過去の製品ラインナップの一部
- SciPOSとの直接競合はキオスク市場よりもドライブスルーAI市場での限定的な重複
- Sam Altman（OpenAI CEO）がエンジェル投資家である点は注目。AI技術への信頼性の裏付け
- 旧Prestoの主要顧客（Applebee's 100,000台タブレット展開等）が現在も継続しているかは要確認
