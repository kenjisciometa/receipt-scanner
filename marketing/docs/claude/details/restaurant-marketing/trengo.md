# Trengo

## 基本情報
- **会社名 / プロダクト名**: Trengo / Trengo（スマート・カスタマーエンゲージメントプラットフォーム）
- **本社所在地**: オランダ ユトレヒト市
- **設立年**: 2017年（創業者：Patrick Meutzner、Marcel van de Weerd）
- **資金調達状況**:
  - 累計調達額：約3,790万ドル
  - 直近ラウンド：2021年9月 シリーズA $36M（Insight Partners、Peak Capital）
- **従業員規模**: 約95〜110名（2025年時点、4大陸に分散）
- **主な展開地域**: ヨーロッパ中心（オランダ、ドイツ、ベルギー、英国など）、北米、アジアへ拡大中

## 料金体系
- **初期費用**: 基本的に不要（SaaSサブスクリプション）
- **月額料金（プラン別）**:

| プラン | 月額（年払い） | 月額（月払い） | 含まれる内容 |
|---|---|---|---|
| **Boost** | €299/月 | €349/月 | ユーザー10名・年間会話数6,000件 |

- **無料プラン / トライアルの有無**: 無料トライアルあり。永久無料プランはなし
- **ハードウェア費用**: 該当なし（クラウドSaaS）

## 機能比較

| 機能カテゴリ | 対応状況 |
|---|---|
| **POSレジ機能** | 非対応（カスタマーコミュニケーションプラットフォーム） |
| **セルフオーダー（キオスク / テーブル / QR）** | 非対応。WhatsApp経由の注文・予約フローは自動化可能 |
| **KDS（キッチンディスプレイ）** | 非対応 |
| **在庫管理** | 非対応 |
| **シフト・勤怠管理** | 非対応 |
| **予約管理** | 間接対応。WhatsApp・メール・チャット経由の予約受付・確認をフローボットで自動化 |
| **CRM / ロイヤルティ** | 連携対応。HubSpot、Salesforce、Pipedrive、Zoho等とネイティブ連携。コンタクト管理機能あり |
| **デリバリー連携** | 間接対応。Shopify連携で注文・配送状況を統合受信トレイで確認可能 |
| **会計ソフト連携** | 限定的。Zapier・n8n等のノーコード連携経由で対応 |
| **レポート・分析** | 対応。パフォーマンス分析、レスポンスタイム、会話解決率、AIエージェント対応率 |
| **多言語対応** | 対応。70以上の言語でAIエージェントが自動応答。メッセージ翻訳機能内蔵 |
| **オフライン対応** | 非対応（クラウドSaaS） |
| **AI / 自動化機能** | 主力機能。AI HelpMate（80%以上の反復会話を自動解決）、Flowbot（ノーコード会話フロー構築）、AI Journeys（2025年）、Smart Suggestions |

**対応チャネル**（統合受信トレイ）: WhatsApp Business API、メール、Instagram Direct、Facebook Messenger、ライブチャット、SMS、音声通話（VoIP）、Telegram

## ハードウェアモデル
- **専用端末 vs BYOD vs 両対応**: ハードウェア非提供。ブラウザ＋モバイルアプリ（iOS/Android）でアクセス
- **対応決済端末**: 非対応
- **プリンタ対応**: 非対応

---

## 強み / 弱み（SciPOS との比較視点）

### 強み
1. **WhatsApp Business APIの公式Metaパートナー**: 98%の開封率を活かした予約確認・プロモーション配信
2. **オムニチャネル統合受信トレイ**: WhatsApp、メール、Instagram、Facebook、SMS、音声を1画面で管理
3. **AI自動化の成熟度**: HelpMate、Flowbot、AI Journeys。コーディング不要で80%以上の反復会話を自動処理
4. **70言語対応のAI**: 多言語市場（欧州）に適した多言語AIエージェント
5. **シンプルなUX**: 非IT人材でも運用しやすい設計思想

### 弱み（SciPOS視点）
1. **POSレジ機能が皆無**: POS、KDS、セルフオーダー、在庫管理、シフト管理を持たない。SciPOSとの競合なし
2. **ハードウェア非対応**
3. **オフライン非対応**
4. **会計・基幹システム連携が限定的**
5. **資金調達が2021年で止まっている**: 大規模プロダクト投資余力に疑問
6. **収益規模がSMB依存**: 2024年売上は$17.9M程度

## 最近の動向（直近1年）

- **2025年3月**: 「Trengo AI」の大型アップデート発表。AI HelpMate、Flowbot、AI Journeysをまとめた統合AIプラットフォームとして再ポジショニング
- **2025年**: AI Journeys正式リリース。ノーコードでカスタム顧客ジャーニーとスマートルーティングを構築
- **2025年**: 複数電話番号チャネル対応。多店舗・多ブランド運営のニーズに対応
- **2025年7月予定**: WhatsApp会話課金モデルの変更（Meta社の料金体系変更に対応）
- **2024年**: 料金体系の全面刷新（シートベース→会話数ベース）
- **2023年9月**: CEO交代。Marili 't Hooft-BolleがCEOに就任。「AIで欧州の顧客エンゲージメントを変革する」戦略を発表
