# Zenchef

## 基本情報
- **会社名 / プロダクト名**: Zenchef（旧称：1001Menus → Zenchef → Zenchef | Formitable グループ）。プロダクト名は **ZenchefOS**
- **本社所在地**: フランス・パリ
- **設立年**: 2010年（1001Menus として創業、2015年に Zenchef へ改称）
- **資金調達状況**:
  - 累計調達額：約 **$58.8M（約€55M相当）**（Crunchbase・4ラウンド、13投資家）
  - 直近の主要ラウンド：2022年9月、PSG Equity より **€50M超** の成長投資を受領。PSG Equity が過半数株主に
  - その後、2023年1月に Formitable（オランダ）と合併、2023年9月に Resengo（蘭・ベルギー）を買収、2024年2月に Tablebooker（ベルギー）を買収
- **従業員規模**: 約 **240名**（パリ、アムステルダム、アントワープ、ブリュッセル、ベルリン、コペンハーゲン、リスボンにオフィス）。2025年7月の CoverManager との統合後は **500名超** の体制へ拡大予定
- **主な展開地域**: フランス、オランダ、ベルギー、ドイツ、デンマーク、スペイン、ポルトガル、オーストリアなど欧州20カ国以上。CoverManager 統合後はスペイン・ラテンアメリカにも拡大
- **顧客規模**: 加盟レストラン **約36,000店舗**（CoverManager 統合後）、月間予約数 **約620万件**
- **年間売上（推定）**: 約 **$24.3M**（2024年10月、Latka 調査）

---

## 料金体系
- **初期費用**: 非公開（問い合わせ必須）
- **月額料金**: **3プラン構成**（Essential / Pro / Enterprise 相当）。月払い・年払いを選択可。具体的金額は非公開で要見積もり
- **課金単位**: 店舗単位（マルチベニュー向けグループプランも存在）
- **無料プラン / トライアル**: 無料プランは存在しない。デモ申込みによる個別提案が基本
- **手数料モデル**: 予約に対する **コミッション完全ゼロ**。TheFork や OpenTable と異なり予約1件あたりの課金なし（最大の差別化ポイント）
- **ハードウェア費用**: 独自ハードウェアは販売していない

---

## ターゲット市場
- **対応業態**: フルサービスレストラン（カジュアル〜ファインダイニング）が中心。ビストロ、ブラッスリー、ミシュラン星付きレストラン、ホテルレストランも対応。QSR・カフェ・バーは主軸ではない
- **対象規模**:
  - 個人・独立系レストランが主要ターゲット（コミッションゼロが訴求点）
  - 中規模チェーン・グループ（マルチベニュー機能で対応）
  - エンタープライズ向けプランも提供
- **主要地域**: 北欧・西欧中心（フランス、ベネルクス、ドイツ、スカンジナビア）。CoverManager 統合後は南欧・ラテンアメリカへ拡大

## 機能比較

| 機能 | 有無・詳細 |
|---|---|
| **POS レジ機能** | なし。Lightspeed、Oracle MICROS Simphony、Trivec、Cashpad、LEO2、Popina、Zelty、unTill、SumUp 等の POS と双方向連携 |
| **セルフオーダー（キオスク / テーブル / QR）** | QR コードによるテーブル会計・支払いに対応（ゲストが QR をスキャンして精算・チップ・割り勘が可能）。キオスク自己注文は非主力 |
| **KDS（キッチンディスプレイ）** | 非対応（POS 側の機能に委譲） |
| **在庫管理** | 非対応 |
| **シフト・勤怠管理** | 非対応 |
| **予約管理** | コア機能。ウィジェット、Google Reserve、Zenchef ゲストアプリ経由の予約、ウェイトリスト、テーブルプラン、プリペイメント・デポジット、グループ予約、電話予約のコール・アシスト機能、マルチベニューウィジェット |
| **CRM / ロイヤルティ** | 強力なゲストプロファイル CRM。予約履歴・アレルギー・嗜好・誕生日・VIP タグ等。ゲスト細分化、メール・SMS キャンペーン、ポイントベースのロイヤルティプログラム。データはレストランが完全所有 |
| **デリバリー連携** | 限定的。クリック＆コレクト（テイクアウト注文）機能あり |
| **会計ソフト連携** | 対応（Mollie 決済連携あり） |
| **レポート・分析** | 予約ソース別トラッキング、稼働率分析、ゲスト行動分析、リアルタイムダッシュボード |
| **多言語対応** | 対応。欧州複数言語での UI・コミュニケーションに対応 |
| **オフライン対応** | クラウドベースのため基本的にオンライン前提 |
| **AI / 自動化機能** | 2025年強化中。AI 自動返信、パーソナライズドプロモーション、自動フォローアップ。CoverManager 統合後は AI 主導のレベニューマネジメントへ進化予定 |

## ハードウェアモデル
- **端末モデル**: BYOD が基本。専用端末の販売・提供なし
- **対応デバイス**: タブレット（iPad 等）、スマートフォン、PC ブラウザ
- **決済端末**: Zenchef 自体は決済端末を提供しないが、Mollie、SumUp 等と連携
- **プリンタ対応**: 非明示（POS 側プリンタを活用）
- **ゲストアプリ**: iOS / Android 向けの Zenchef ゲストアプリを提供

## 強み / 弱み（SciPOS との比較視点）

### 強み
1. **コミッションゼロの予約モデル**: TheFork・OpenTable と異なり予約課金なし。独立系レストランに高い訴求力
2. **欧州最大規模の直接予約ネットワーク**: 36,000店舗・月620万予約（CoverManager 統合後）
3. **強力な CRM・マーケティング機能**: ゲストデータの完全所有、セグメント別配信、ロイヤルティプログラムが一体化
4. **PSG Equity による豊富な資本と M&A 戦略**: Formitable、Resengo、Tablebooker、CoverManager を次々と統合
5. **POS 非依存モデル**: 主要 POS ベンダーと連携可能。既存 POS を入れ替える必要がない

### 弱み（SciPOS が優位な点）
1. **POS レジ機能が存在しない**: 注文受付・会計・キッチン管理は対象外。SciPOS はオールインワン POS として競合しない
2. **QSR・カフェ・テイクアウト業態への適合性が低い**: フルサービスレストランの席予約前提
3. **キオスク・セルフオーダー・KDS が非対応**
4. **在庫管理・シフト管理が非対応**
5. **価格非公開・サポート品質の課題**: Trustpilot・Glassdoor で否定的レビューも散見
6. **アジア市場への展開なし**: 欧州・ラテンアメリカが主戦場

## 最近の動向（直近1年）

- **2024年1月**: Formitable・Resengo・旧 Zenchef の3社が統一ブランド「Zenchef」として完全統合。ZenchefOS へのプラットフォーム統合を発表
- **2024年2月**: ベルギーの Tablebooker を買収。欧州での加盟店数19,000店舗・月620万予約に到達。欧州最大のコミッションフリー予約プラットフォームを宣言
- **2024年通年**: ZenchefOS に多数の機能追加。マルチベニューウィジェット、スマートスケジューリング、コールアシスト、グループ予約管理、テーブルプラン最適化など
- **2025年初頭**: CEO Thomas Jeanjean がAI主導のゲストエンゲージメント、UI/UXへの大規模投資、ローカルサポート強化を年間優先事項として発表
- **2025年7月**: スペインの CoverManager との合併を発表。統合後グループは500名超、10カ国以上で展開、計36,000店舗超。AI 主導のレベニューマネジメントプラットフォームへの転換を目指す
- **2026年2月**: 自動 SMS 予約機能、マルチベニューウィジェット、スケジュール管理画面の刷新をリリース
