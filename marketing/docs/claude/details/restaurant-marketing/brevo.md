# Brevo

## 基本情報
- **会社名 / プロダクト名**: Brevo（旧称：Sendinblue）
- **本社所在地**: フランス・パリ17区。その他、デリー（インド）、シアトル、ベルリン、ソフィア、トロント、ニューヨーク、ウィーン の計8拠点
- **設立年**: 2012年（創業者：Armand Thiberge）
- **資金調達状況**:
  - 累計調達額：約7億8,000万米ドル（4ラウンド）
  - 2020年10月：Series B 1億6,000万ドル（Bridgepoint・Bpifrance・BlackRock主導）
  - 2025年12月：Series C 5億8,300万ドル（€500M相当）―General Atlantic・Oakley Capital参加。ユニコーン到達（評価額10億ドル超）
- **従業員規模**: 約900〜1,000名以上（68以上の国籍）
- **主な展開地域**: ヨーロッパ（フランス・ドイツ・東欧）、北米、インド。顧客数60万社以上

## 料金体系
- **初期費用**: なし
- **月額料金（プラン別）**:

| プラン | 月額（税別・概算） | 主な特徴 |
|---|---|---|
| **Free** | 無料 | 1日300通メール・無制限コンタクト・基本CRM |
| **Starter** | 約$9〜 | メール送信数に応じスケール・オートメーション2,000コンタクト上限 |
| **Business** | 約$18〜$499〜 | 高度なオートメーション・LP・WhatsApp・電話サポート |

- **無料プラン / トライアル**: 永続無料プランあり（1日300通・コンタクト無制限・CRM・基本オートメーション）
- **ハードウェア費用**: なし（SaaSのみ）

## 強み / 弱み（SciPOS との比較視点）

### 強み
1. **圧倒的コストパフォーマンス**: 無料プランでCRM・メール・基本オートメーションが利用可能。送信数課金のため大規模リストでも割安
2. **オールインワン顧客エンゲージメント**: メール・SMS・WhatsApp・プッシュ・ライブチャット・CRM・ロイヤルティを1プラットフォームに集約
3. **飲食業のロイヤルティ実績**: 50万ロイヤルティカードデジタル化事例
4. **GDPR対応・欧州ファースト**: B Corp認証取得（2025年5月・スコア130.5）
5. **急成長・資金力**: 2025年にユニコーン到達、ARR 2億ユーロ超見込み
6. **AI機能の先進性**: MCP対応でClaudeやChatGPTと連携

### 弱み
1. **POSとは競合しない**: SciPOS が提供するPOS・KDS・セルフオーダー・シフト管理等を一切持たない（補完的ツール）
2. **テンプレート・デザイン機能が限定的**: Mailchimp・Klaviyoと比較して劣る
3. **分析の深さがKlaviyoに劣る**: eコマース向け高精度セグメント・LTV分析等
4. **Starterプランのオートメーション制限**: 対象コンタクト2,000件上限
5. **WhatsApp機能の制限**: Professional・Enterprise プランのみ
