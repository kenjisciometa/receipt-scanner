# Partoo

## 基本情報
- **会社名 / プロダクト名**: Partoo（パルトゥー）
- **本社所在地**: フランス・パリ
- **設立年**: 2014年7月（創業者：Thibault Levi-Martin）
- **資金調達状況**:
  - 2019年3月：フランスの大手デジタルメディアグループ **Webedia** に買収される
  - 2021年5月：Webediaより **1,500万ユーロのシリーズB** 追加出資
  - 2024年時点のバリュエーション：約 **9,000万USD**
  - 2024年のARRは **3,550万ユーロ超** を達成し、2025年初頭に黒字転換
  - 2025年末までに **ARR 1億ユーロ（"Apollooプラン"）** を目標に設定
- **従業員規模**: 約 **350〜495名**
- **主な展開地域**: フランス、スペイン、イタリア、ポルトガル（欧州南部）、中東、ラテンアメリカ（メキシコ注力）。売上の **50%超が海外**。100カ国以上でサービス展開

## 強み / 弱み（SciPOS との比較視点）

### 強み
1. **ローカルSEO・オンラインプレゼンスの専門性**: Google、Apple Maps、Waze等への一括配信で欧州・ラテンアメリカで高シェア
2. **大規模チェーン向け実績**: McDonald's、Carrefour、Toyota等の店舗網を管理
3. **AIエージェント「Jim」**: 問い合わせの70〜80%を自動処理。Cash Converters Belgiumでの実証済み
4. **高い顧客維持率**: エンタープライズ更新率90%、NPS 50超
5. **収益性**: 2025年初頭に黒字転換。外部資本に依存しない持続可能成長モデル
6. **レビュー管理の深度**: Google・Facebook・TripAdvisorの一元管理、AI返信自動化、競合分析

### 弱み（SciPOSとの比較視点）
1. **POSシステムではない**: レジ・注文・会計・KDS・在庫・シフト管理を一切持たない。SciPOSとは異なるカテゴリ
2. **デリバリー・自社オーダー連携なし**
3. **ハードウェア非対応**
4. **価格の透明性の低さ**: 中小企業が事前に費用感を把握しにくい
5. **エンタープライズ偏重**: 個人飲食店には機能過多・コスト高の可能性
