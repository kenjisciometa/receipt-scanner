## 料金体系

Flatpayの最大の差別化は**フラットレート（均一取引手数料制）**。月額サブスクリプション料金なし、初期費用なし、隠れコストなし、という透明性を訴求。

| 月額費用 | **€0** |
| 初期費用 | **無料**（ハードウェア費用は別途） |

### 共通
- **隠れ費用**: なし（Flatpayの最大の訴求点）
- **日次精算（Daily Payouts）**: 翌営業日払い
- **24/7カスタマーサポート**: 全プランに含む（追加費用なし）
- **オンサイト設置**: ハードウェア・ソフトウェアの訪問セットアップを提供
- **無料トライアル**: 非公開（要問合せ）

### フィンランド市場での価格ポジション
フィンランドでは「**hintahaastaja**（価格チャレンジャー）」として市場参入。Nets/Verifone等の既存プレイヤーに対して、0.99%のフラットレートと隠れコストゼロを武器に差別化。ALV（付加価値税）率変更に対応したPOSシステムも訴求ポイント。

## 販売モデル（重要な差別化要素）

Flatpayの最大の特徴の一つが**ドア・ツー・ドア（訪問型）セールスモデル**:

- 営業担当者がSMBを直接訪問し、ペンと紙で料金説明
- その場でデモ・即時契約が可能
- Stripe/SumUp/PayPal等のデジタルセルフサービス型と対照的
- 顧客獲得コスト（CAC）は高いが、コンバージョン率・リテンション率で補填
- 1,500名の従業員の多くがフィールドセールス要員
- 2026年末に3,000名（倍増）を計画し、さらなる営業力強化

**フィンランドでの営業**: フィンランド市場でも同様の訪問型セールスを展開。現地語サポートと組み合わせて、既存のNets/Verifone加盟店からの切り替えを積極的に獲得。

## 強み / 弱み（SciometaPOS との比較視点）

### 強み
1. **フラットレート x ゼロ月額の破壊的価格**: 従来のNets/Worldline/Adyen比で料金の透明性・シンプルさで優位。SMBの「決済コストが読めない」不安を解消。フィンランドでは「hintahaastaja」として明確にポジショニング
2. **圧倒的な成長速度**: 設立3年でユニコーン（デンマーク最速）。7,000 → 60,000〜70,000加盟店を1年強で達成。ARR €1M/日の増加ペース
3. **資本力 $239M（累計）**: Series C $170Mにより欧州展開・技術投資・採用（3,000名目標）を積極的に実行可能
4. **訪問型セールスの威力**: ドア・ツー・ドアでSMBに直接リーチ。デジタルマーケティング依存の競合と差別化。高コンバージョン・高リテンション
5. **オンサイト設置 + 24/7サポート**: ハードウェア含めた訪問セットアップはSMBの技術障壁を下げる。全プラン24/7サポート無料
6. **フィンランド市場で急成長中**: 2,500+加盟店を獲得。フィンランド語完全対応。ALV変更対応でタイムリーな乗り換え促進
7. **統合決済プラットフォームへの進化**: 店内POS + オンライン決済 + 計画中のバンキングスイートで加盟店の全金融ニーズを囲い込み

### 弱み
1. **機能がシンプルすぎる（機能の広さ不足）**: セルフオーダー（QR/キオスク）、シフト管理、予約、CRM、デリバリー連携がなく、SciometaPOSの統合ソリューションには機能面で大きく劣る
2. **レストラン特化機能が限定的**: テーブル管理・分割会計はあるが、KDS・メニューエンジニアリング・在庫管理・ロイヤルティが弱い。フルサービスレストランへの対応は浅い
3. **専用ハードウェア必須・BYOD非対応**: 導入の柔軟性が低く、既存POS端末の流用ができない。ハードウェア費用（€1,495〜€2,495）が初期障壁
4. **未黒字化**: 高CACの訪問型セールス + 300%成長投資のため利益が出ていない。資金枯渇リスクあり（ただし€145M調達済み）
5. **AI機能は限定的・後発**: AI在庫予測は初期段階。CEOがAIに懐疑的な発言をしており、AI投資の優先度は低い
6. **フィンランドでの会計ソフト連携が不透明**: Procountor/Netvisorとの直接連携が確認できない。フィンランドのレストランにとってこれは導入障壁になりうる
7. **POS手数料が1.49%**: POSプランの取引手数料は0.99%ではなく1.49%（年間€200K未満の場合）。ターミナル単体の0.99%と比べて高い

### SciometaPOSとの競合関係

| 比較軸 | SciometaPOS | Flatpay |
|--------|-------------|---------|
| POSレジ | ○ | ○ |
| セルフオーダー | ○（QR + キオスク） | **なし** |
| KDS | ○ | 限定的（キッチンプリンタのみ） |
| シフト管理 | ○ | **なし** |
| 在庫管理 | ○ | 限定的 |
| CRM/ロイヤルティ | ○ | **なし** |
| デリバリー連携 | ○ | **なし** |
| 決済処理 | 外部連携 | **自社決済（コア事業）** |
| BYOD | ○ | **なし** |
| AI機能 | ○（開発中） | 限定的 |
| ハードウェア | 汎用タブレット | 専用端末（CPOS X5等） |
| 月額費用 | サブスクリプション | 無料（取引手数料のみ） |
| フィンランド市場 | 展開検討中 | **2,500+加盟店** |

**競合の核心**: Flatpayは「決済 + シンプルPOS」、SciometaPOSは「統合レストラン運営プラットフォーム」。ターゲット層は重なる（欧州SMBレストラン）が、価値提供の方向性が異なる。

**フィンランドでの競合シナリオ**:
- Flatpayはフィンランドで既に2,500+加盟店の基盤を持つ先行者
- SciometaPOSがフィンランドに進出する場合、Flatpay加盟店の一部は「機能不足」を感じてSciometaPOSに乗り換える可能性がある（特にフルサービスレストラン）
- 逆に、Flatpayの訪問型セールス + 0.99%フラットレートは、SciometaPOSの市場獲得を困難にする可能性（価格競争力）
- **連携の可能性**: FlatpayはAPI連携に積極的。SciometaPOSがFlatpayを決済ゲートウェイとして採用する形での連携は理論上可能。ただしFlatpayがPOS機能を拡張中のため、長期的にはパートナーよりも競合になる方向

## SciometaPOSへの示唆

1. **フィンランド市場進出時はFlatpayが最重要競合**: 2,500+加盟店の基盤と訪問型セールスの営業力、フィンランド語完全対応は脅威。SciometaPOSはFlatpayにない「統合機能」（セルフオーダー、KDS、シフト管理、在庫管理）で差別化する必要がある
2. **価格競争は避けるべき**: Flatpayの0.99%フラットレート+月額ゼロは価格破壊的。SciometaPOSは「ROI」（人件費削減、売上増、効率化）で価値を示す戦略が有効
3. **Flatpayの機能拡張を注視**: バンキングスイート・BNPL・AI分析の追加でFlatpayの付加価値が上がる可能性。ただし現時点ではレストラン特化機能は弱い
4. **連携 vs 競合の判断**: 短期的にはFlatpayを決済パートナーとして連携する選択肢もある。長期的にはPOS機能拡張で直接競合になるリスクが高い
5. **Procountor/Netvisor連携の優位性**: Flatpayがフィンランドの会計ソフト連携に弱い場合、SciometaPOSがこれを先に実装することで差別化ポイントになる
