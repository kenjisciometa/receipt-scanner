## 料金体系

課金単位は**最初の30ユーザーまで定額 + 31人目以降は1ユーザーあたり/月の従量課金**。3つのハブ（Operations / Communications / HR & Skills）ごとに異なるプランを選択可能。

| プラン | 月額（月次払い） | 月額（年次払い） | 主な機能 |
|--------|----------------|----------------|---------|
| Small Business（無料） | $0 | $0 | 10ユーザーまで。各ハブの基本機能を利用可能 |
| Basic | $35/月（30名まで） | $29/月（30名まで） | スケジュール管理、タイムクロック、フォーム、チャット、基本レポート |
| Advanced | $59/月（30名まで） | $49/月（30名まで） | Basic + ジオフェンシング、自動シフト割当、高度フォーム、スマートグループ |
| Expert | $119/月（30名まで） | $99/月（30名まで） | Advanced + ライブGPS追跡、プロセス自動化、自動翻訳、マルチブランチ管理 |
| Enterprise | カスタム | カスタム | Expert + SSO、API、専任CSM、カスタム連携、監査ログ |

- **31人目以降の追加料金**: Basic $0.5/月/人、Advanced $1.5/月/人、Expert $3/月/人（年次払い時、概算）
- **年間払い割引**: 約18% OFF
- **無料プラン**: あり（Small Business Plan、10ユーザーまで永久無料）
- **無料トライアル**: 14日間（全有料機能を利用可能）
- **ハードウェア費用**: なし（SaaS専業、BYOD）

## ターゲット市場

### 対応業態
ノンデスクワーカー全般を対象。レストラン・飲食店、小売、建設、清掃、セキュリティ、物流、フィールドサービス、医療・介護、製造業など幅広い業種に対応。飲食業専業ではなく汎用型。

### 対象規模
- **小規模事業者（1〜10名）**: 無料プランで獲得。PLG（Product-Led Growth）戦略の入口
- **SMB（10〜200名）**: 主力ターゲット。有料プランの大半がこの層
- **ミッドマーケット〜エンタープライズ（200名以上）**: Enterpriseプランで対応。SSO・API・専任CSMを提供

### 主要地域
米国（最大市場）、英国、オーストラリア、カナダ、イスラエル。欧州・中南米にも展開。日本・アジア市場への本格進出は未確認。

## 強み / 弱み（SciometaPOS との比較視点）

### 強み
1. **圧倒的な機能幅（オールインワン）**: シフト管理・打刻・コミュニケーション・タスク管理・トレーニング・HRを1アプリに統合。ノンデスクワーカーのあらゆるニーズをカバー。競合の7shiftsやSkelloがシフト管理専業なのに対し、Connecteamは従業員エンゲージメント全体をカバー。
2. **無料プランの強力な誘引力**: 10ユーザーまで永久無料。小規模飲食店が無料で導入→成長に伴い有料化するPLGモデルが強力。
3. **価格競争力**: 30ユーザーまで$29〜99/月（年次払い）は、ユーザー単価で見ると業界最安クラス。特に20〜30名規模でのコスパが際立つ。
4. **GPS / ジオフェンシングの深い実装**: ブレッドクラムGPS追跡、ジオフェンス半径のカスタマイズ、セルフィー認証など不正打刻防止が充実。フィールドワーカー管理に強い。
5. **大規模な資金調達と成長**: $160M以上の資金調達、$800M+バリュエーション、500名超の従業員。プロダクト開発・グローバル展開の継続力がある。
6. **AIエージェントの早期実装（2026年）**: 従業員向けAIエージェントをリリースし、ナレッジベースからの自動回答で管理者負担を軽減。

### 弱み
1. **飲食業特化ではない**: 汎用型のため、飲食業固有の機能（ティップ管理、食品安全コンプライアンス、メニュー連動シフト最適化等）は薄い。7shiftsやSkelloの飲食特化には及ばない。
2. **POS / 決済 / 在庫 / セルフオーダーがない**: SciometaPOSが提供する統合POS体験との直接競合はないが、飲食店が「POSもシフトも1つのシステムで」と求める場合には選ばれない。
3. **労働法コンプライアンスの地域深度が浅い**: 欧州や日本の業種別労働協定（Convention collective等）への深い対応はSkelloほどではない。グローバル展開だが各国ローカル規制への対応は限定的。
4. **機能過多による複雑さ**: オールインワンゆえに設定・管理が複雑になりがち。飲食店オーナーが「シフト管理だけ使いたい」場合でも3ハブの構造を理解する必要がある。
5. **日本語対応なし**: 日本市場向けのローカライゼーション（日本語UI、日本の労働基準法対応、日本の給与ソフト連携）は未実装。
6. **Lightspeed POS以外の飲食POSとの連携が限定的**: POS連携はLightspeedのみが公式。Square、Toast、SciometaPOS等との連携は未確認。

### SciometaPOSとの競合関係
- **直接競合: 部分的**（SciometaPOSのシフト管理モジュールとConnecteamのOperationsハブが重複）
- **差別化ポイント**: SciometaPOSは「POS + セルフオーダー + KDS + 在庫 + シフト管理」の飲食業統合プラットフォーム。Connecteamは「シフト + 打刻 + コミュニケーション + HR + タスク」のノンデスクワーカー汎用プラットフォーム。アプローチが根本的に異なる。
- **脅威レベル: 中**: Connecteamの無料プランと価格競争力は、特に小規模飲食店がシフト管理だけを求める場合に脅威。ただしPOS統合の価値を訴求できれば差別化可能。
- **連携の可能性**: ConnecteamのAPI（Enterpriseプラン）を通じた連携は技術的には可能だが、戦略的には競合関係にあるため優先度は低い。

## 最近の動向（直近1年）

1. **AIエージェント機能リリース（2025年〜2026年）**: 従業員がAIエージェントに質問すると、ナレッジベースから自動回答する機能を実装。2026年2月にはAIエージェント利用状況トラッキング（Usage tab）とカスタムインストラクション機能を追加。管理者がAIの応答方針を制御可能に。
2. **業界初のヘルプデスク機能（2025年）**: ノンデスクワーカー向けの社内ヘルプデスクをリリース。オンボーディング、機器故障、給与に関する内部サポートリクエストをモバイルから一元管理。
3. **Accu-Time Systems との戦略的統合（2026年3月）**: ハードウェア打刻端末メーカーとの統合により、物理タイムクロックとConnecteamアプリの連携を実現。
4. **従業員規模500名突破（2026年）**: 2022年Series C時点から継続的に採用拡大。プロダクト・エンジニアリング部門を中心に増員。
5. **ニュースフィードの機能強化**: カスタマイズ可能なニュースフィード、既読追跡、フォーム更新時の自動チャット通知など、コミュニケーションハブの改善を継続。
6. **追加資金調達なし**: 2022年3月のSeries C（$120M）以降、新規ラウンドの発表なし。収益性改善またはIPO準備段階と推測される。
