## 料金体系

Harriはエンタープライズ向けカスタム見積りが中心。公開料金は限定的。

| 項目 | 内容 |
|------|------|
| 初期費用 | 導入・実装費あり（例: Harri IQモジュールで約$625。規模により変動） |
| 月額料金 | カスタム見積り（例: 100名/10拠点でHarri IQ $375/月。モジュール追加で増額） |
| 課金単位 | 従業員数ベース＋モジュール別（ヘッドカウント閾値・拠点数で段階的料金設計） |
| 無料プラン | なし |
| 無料トライアル | なし（デモリクエスト制） |
| ハードウェア費用 | 顔認証タイムクロック等の専用デバイスが別途必要な場合あり |

**備考**: 中小規模のホスピタリティ事業者にとっては割高との評価が多い。大規模マルチロケーション企業向けに最適化された価格設計。

### 主要AI機能詳細（Harri Agentic AI）

2025年9月にオープンベータとしてローンチ。「Salli」というエージェントが複数のAIエージェントを統合して提供。

- **スマートスケジューリング**: 従業員の可用性、需要予測、労働法を自動考慮した公平でコンプライアント対応のロスター自動生成
- **需要予測**: 過去のPOS売上データ、天気予報、イベントカレンダー、SNSトレンドを統合分析し、来店需要を予測
- **常時コンプライアンス保護**: 休憩漏れ・法令違反リスクを罰金発生前に自動検出・警告
- **リアルタイムガイダンス**: 労務コスト最適化の即時サポート、意思決定疲労の軽減

### 採用・HCM機能（シフト管理以外のコア機能）

Harriは30以上のモジュールを持つフルスタックHCMプラットフォーム:

- **採用・タレントアクイジション**: 採用ブランディング、ATS（応募者追跡）、求人配信
- **オンボーディング**: デジタルオンボーディング、書類管理、コンプライアンス確認
- **従業員エンゲージメント**: グループメッセージング、アナウンス、フィードバック収集
- **パフォーマンス管理**: 従業員評価、スキルトラッキング
- **コンプライアンス**: Fair Workweek法、児童労働法、残業規制、休憩管理の自動適用

## 強み / 弱み（SciometaPOS との比較視点）

### 強み
1. **エンタープライズ規模の実績**: McDonald's、Shake Shack、Subway等の大手チェーンが採用。55,000拠点・400万従業員の実績は業界トップクラス。
2. **フルスタックHCM**: 採用→オンボーディング→スケジューリング→勤怠→エンゲージメント→退職までの従業員ライフサイクル全体をカバー。シフト管理単体ではなく包括的なHCMとして差別化。
3. **Agentic AIの先行実装**: 天気・イベント・SNSトレンドまで考慮した需要予測AIは業界最先端。2025年9月ベータ開始で先行者優位を確保。
4. **ADP戦略的パートナーシップ**: 2025年にADPからの投資・統合提携を実現。ADP Workforce Now連携により給与処理のシームレス化が完了。
5. **$149.1M売上・高成長率**: 2024年に前年比+37%成長。Series B後もオーガニック成長を維持し、資本効率が高い。
6. **米国労働法コンプライアンスの深度**: Fair Workweek法、残業規制、児童労働法など米国特有の複雑な労働規制に深く対応。

### 弱み
1. **POS/決済/在庫/セルフオーダーがない**: HCM/WFM専業のため、レストランの業務オペレーション全体はカバーしない。SciometaPOSのような統合プラットフォームとは設計思想が異なる。
2. **中小規模には割高**: エンタープライズ向け価格設計のため、個人店や小規模チェーンには非現実的。
3. **UXの成熟度**: ユーザーレビューでUIが「やや粗い」「200拠点以上のオンボーディングが困難」「バグ・グリッチがある」との指摘。通知の不安定さ、再ログイン頻発の不満あり。
4. **欧州・アジア展開が限定的**: 英国には進出済みだが、EU大陸やアジア市場向けの労働法対応・多言語対応は未成熟。日本市場への展開は確認できず。
5. **無料プラン/トライアルなし**: デモリクエスト制のため、小規模事業者の導入ハードルが高い。

### SciometaPOSとの競合関係
- **直接競合: なし**（HarriはHCM/WFM専業、SciometaPOSはPOS+セルフオーダー+シフト管理の統合プラットフォーム）
- **間接競合**: SciometaPOSのシフト管理モジュールとは機能的に重複するが、ターゲット規模（Harri=エンタープライズ、SciometaPOS=中小〜中堅）と地域（Harri=米英中心、SciometaPOS=日本・アジア）が大きく異なる
- **差別化ポイント**: SciometaPOSはPOS・セルフオーダー・KDS・在庫・シフト管理を1プラットフォームで提供する「統合型」。Harriは採用からエンゲージメントまでの「従業員ライフサイクル全体型」。設計思想が根本的に異なる
- **学ぶべき点**: HarriのAI需要予測（天気・イベント・POS売上データの統合分析）はSciometaPOSのシフト管理モジュールへのAI機能強化の参考になる

## 最近の動向（直近1年）

1. **Agentic AI オープンベータ開始（2025年9月）**: 「Salli」エージェントを中心としたAgentic AIプラットフォームをローンチ。需要予測スケジューリング、コンプライアンスリスク自動検出、リアルタイム労務最適化を提供。ホスピタリティ業界向けAgentic AIとして先駆的ポジション。
2. **ADP投資・統合提携（2025年2月）**: ADP Venturesからの追加投資と、ADP Workforce Nowとの統合ホスピタリティ体験のローンチを発表。給与処理とワークフォース管理のシームレス連携を実現。
3. **ADP統合ホスピタリティ体験ローンチ（2025年11月）**: HarriのAI駆動ワークフォース管理とADP Workforce Now給与処理を統合した新サービスを正式発表。
4. **売上$149.1M達成（2024年）**: 前年比+37%の高成長を維持。顧客数380社、55,000拠点へ拡大。
5. **MURTEC 2026スポンサー**: 2026年3月のMURTEC（Multi-Unit Restaurant Technology Conference）のスポンサーとして参加。エンタープライズレストラン市場でのプレゼンス強化。
