## 料金体系

課金単位は**ロスター（roster）単位/月**。1ロスターは最小10ユーザーを含む。通貨は SGD（シンガポールドル）。

| プラン | 月額 | 主な機能 |
|--------|------|---------|
| Start Up（無料） | SGD 0 | 設立1年以内・25名以下の企業向け。スケジューリング、打刻（Clock-in）、タイムシート基本機能 |
| Basic | SGD 119/ロスター | スケジューリング、タイムクロック（QR / ジオフェンス / Wi-Fi）、タイムシート自動集計、基本レポート |
| Growth | SGD 129/ロスター | Basic 全機能 + マルチアウトレット管理、無制限スタッフ、高度な自動スケジューリング、詳細レポート |
| Pro | SGD 129/ロスター | Growth 全機能 + Mass Grab Shift、高度なタイムシートルール、高度な休暇管理、データ分析、コスト予測・推定、Direct API アクセス |
| Enterprise | 要問い合わせ | 100名超対象。全機能、カスタム連携、優先サポート |

- **初期費用**: 非公開（オンサイトトレーニングは SGD 600/2時間セッション）
- **無料プラン**: Start Up プラン（設立1年以内・25名以下限定）
- **無料トライアル**: あり
- **ハードウェア費用**: 非該当（SaaS 専業。打刻用タブレット/スマホは BYOD）

## ターゲット市場

### 対応業態
F&B（レストラン、カフェ、バー、QSR）、リテール（小売）。飲食業が最重点ターゲット。

### 対象規模
- **スタートアップ / 個人店**: Start Up プラン（無料）で対応
- **SME（中小規模チェーン）**: Basic〜Pro プランが主力
- **エンタープライズ（100名超）**: Enterprise プランで対応

### 主要地域
シンガポール（最大市場）、マレーシア（staffany.my で展開）、インドネシア。東南アジア ASEAN 圏に集中。

## 強み / 弱み（SciPOS との比較視点）

### 強み
1. **F&B シフト管理への深い特化**: 飲食業のシフトワーカー管理に完全特化しており、業界固有の課題（時間詐取、早期打刻、不要な残業）に対する具体的ソリューションを提供
2. **無料プランの存在**: スタートアップ向け無料プラン（25名以下・設立1年以内）があり、小規模店舗の獲得に有利
3. **東南アジア ASEAN 市場での先行**: シンガポール、マレーシア、インドネシアでの展開実績。東南アジアの労務規制（CPF、外国人労働者枠等）への対応
4. **直感的な UI**: ユーザーレビューで「技術に不慣れな従業員でも使いやすい」と高評価。導入障壁が低い
5. **労務コスト最適化**: 最大30%の労務コスト削減を謳う。リアルタイム予算追跡、タイムシート自動化により給与処理を「3日から3時間」に短縮したユーザー事例あり
6. **ロスター単位の課金**: ユーザー数ではなくロスター単位の課金のため、大人数の店舗ではコスト効率が良い
7. **現地給与ソフト連携**: シンガポール・マレーシアの給与ソフト（Talenox、Payboy 等）との連携が充実

### 弱み
1. **WFM 単機能のみ**: POS、セルフオーダー、KDS、在庫管理、予約管理、CRM 等の機能が一切なく、レストラン運営の統合プラットフォームとしては不完全。SciPOS のようなオールインワン戦略とは対極
2. **地域限定性**: シンガポール・マレーシア・インドネシアに集中。日本・欧州市場への展開なし。日本語非対応
3. **小規模な資金基盤**: 累計約 $5.9M の調達は競合（7shifts: $100M+、Planday: 買収済み等）と比較して著しく小さく、機能開発・市場拡大の速度に制約
4. **連携パートナーの限定性**: POS 連携は Novitee、Waffle のみ。主要グローバル POS（Square、Toast、Lightspeed 等）との連携なし
5. **エンタープライズ対応の弱さ**: 46名体制での大規模チェーン対応は困難。Enterprise プランの詳細が非公開
6. **オフライン対応不明**: クラウド専業のため、ネットワーク不安定な環境での運用に懸念
7. **ユーザー動向の懸念**: 一部レビューサイトでユーザートレンドの低下が指摘されている

### SciPOS との差別化ポイント
- SciPOS はPOS・セルフオーダー・KDS・在庫・シフト管理を統合したオールインワンプラットフォーム。StaffAny はシフト管理単機能のため、レストランは別途 POS を導入する必要があり、データのサイロ化が発生する
- SciPOS は日本・欧州市場をターゲットとしており、StaffAny の東南アジア ASEAN 圏とは市場が異なる。直接競合よりも参考事例としての位置づけ
- StaffAny の AI 自動スケジューリングやコスト予測機能は、SciPOS のシフト管理モジュールの機能拡張の参考になる

## 最近の動向（直近1年）

- **2025年**: シンガポール F&B 業界向けに「2025 HR ベンチマーク・トレンドレポート」（給与ベンチマーク、採用プラットフォーム、研修予算等）を発行。F&B 業界のナレッジハブとしてのブランディングを強化
- **2025年**: マレーシア市場向けに外国人労働者雇用ガイド等のコンテンツマーケティングを展開。staffany.my ドメインで現地化を推進
- **2024〜2025年**: Progressive Wage Model（PWM、シンガポール政府の段階的賃上げ制度）への対応コンテンツを積極的に発信。規制対応をセールスポイントとして活用
- **2023年3月**: Non-equity Assistance ラウンドを実施（直近の資金調達）
- **大型の追加資金調達や買収のニュースは確認されず**: 2022年の Series A ($3.4M) 以降、新たなエクイティラウンドの公開情報なし。比較的慎重な成長戦略を取っている模様
