## 料金体系

課金単位は**従業員ライセンス数/月**。月間プランは日割り計算で請求。

| プラン | 月額 | 主な機能 |
|--------|------|---------|
| Basic | 無料（20名未満の新規導入企業） | 基本スケジューリング、打刻、勤怠記録 |
| Standard | 約 $4/人/月（公開価格） | Basic 全機能 + 各種勤怠要請（勤務日程修正・休暇申請等）、カスタムリクエスト、休暇促進システム、メッセージ機能、自動休暇付与 |
| Enterprise | 要問い合わせ（100名以上向け） | Standard 全機能 + 電子決裁ワークフロー、カスタム運用環境構築、高度なセキュリティ・アクセス制御、専任サポート |

- **初期費用**: 非公開
- **無料プラン**: Basic プラン（20名未満の新規企業限定）
- **無料トライアル**: 30日間（Standard プラン機能を体験可能。クレジットカード不要）
- **ハードウェア費用**: 非該当（SaaS 専業。打刻用デバイスは BYOD）
- **決済方法**: Basic / Standard はアプリ内カード決済。Enterprise は請求書・仮想口座決済対応

## ターゲット市場

### 対応業態
全業種対応型の WFM ソリューション。飲食（レストラン・カフェ）、小売、医療・福祉、物流・製造、コールセンター、宿泊施設など幅広い業種をカバー。特定業種特化ではなく汎用型。

### 対象規模
- **小規模事業者**: Basic プラン（無料・20名未満）で対応。2025年4月には韓国雇用労働部と連携し零細企業への無料提供を実施
- **中堅企業**: Standard プランが主力
- **大企業・エンタープライズ**: Enterprise プランで対応。SK、ロッテ、現代等の韓国財閥系大企業が導入

### 主要地域
韓国（30万社以上の圧倒的シェア）、台湾（2023年〜、開始2ヶ月で有料顧客400社獲得）。今後、シンガポール・日本への展開を計画中。

## 強み / 弱み（SciPOS との比較視点）

### 強み
1. **韓国市場での圧倒的シェア**: 30万社以上の導入実績は韓国 WFM 市場で No.1。SK、現代、ロッテ、カカオ等の大企業から零細企業まで幅広いセグメントをカバー
2. **高い収益性**: 2024年の営業利益率75%、売上倍増・営業利益3倍増と極めて高い成長性と収益性を両立。SaaS ビジネスとして非常に効率的な運営
3. **強力な資本バック**: SkyLake Equity Partners（韓国有力 PE ファンド）が $280M で75%株式を取得。アジア展開のための潤沢な資金基盤を確保
4. **GPS / Wi-Fi 打刻の偽装防止**: 偽造座標・偽造 Wi-Fi の検知機能を標準搭載。不正打刻防止に強み
5. **韓国労働法への深い対応**: 週52時間制度、PC-off 機能、休暇促進制度等、韓国特有の労務規制にフル対応。法令遵守を自動化
6. **大企業 ERP との連携実績**: 韓国財閥系大企業の社内 ERP システムとの統合実績があり、エンタープライズ対応力が高い
7. **低価格帯**: $4/人/月は WFM ソリューションとしてグローバルでも競争力のある価格設定。20名未満は無料

### 弱み
1. **WFM 単機能**: POS、セルフオーダー、KDS、在庫管理、予約管理、CRM 等の飲食店運営に必要な機能が一切なく、レストラン向け統合プラットフォームとしては不完全。SciPOS のオールインワン戦略とは対極
2. **韓国市場への過度な依存**: 30万社の顧客基盤はほぼ全て韓国国内。海外展開は台湾が緒についたばかりで、グローバルでの実績は限定的
3. **従業員規模の制約**: 15名体制で30万社をサポートしており、海外展開に伴うローカルサポート・カスタマイズ対応には制約がある
4. **飲食業特化ではない**: 全業種汎用型のため、飲食業固有の課題（ピーク時間帯の最適配置、売上連動スケジューリング等）への対応は StaffAny や 7shifts 等の F&B 特化型と比較して限定的な可能性
5. **日本市場未参入**: 日本語対応の有無が不明。日本の労働基準法（36協定、変形労働時間制等）への対応実績なし。日本展開は計画段階
6. **AI / 自動スケジューリング機能の限定性**: 自動シフト作成等の高度な AI 機能は、欧米競合（When I Work、7shifts 等）と比較して情報が限定的
7. **POS / 外部システム連携の不透明さ**: グローバル POS システム（Square、Toast、Lightspeed 等）や会計ソフトとの連携情報が確認できず

### SciPOS との差別化ポイント
- SciPOS は POS・セルフオーダー・KDS・在庫・シフト管理を統合したオールインワンプラットフォーム。Shiftee は WFM 専業のため、飲食店は別途 POS を導入する必要がありデータのサイロ化が発生する
- Shiftee は韓国市場で圧倒的なシェアを持つが、日本・欧州市場への展開は未着手。SciPOS が先行する日本・欧州市場では当面直接競合しない
- Shiftee の GPS/Wi-Fi 偽装防止打刻機能や韓国労働法への自動対応は、SciPOS のシフト管理モジュール強化の参考になる
- Shiftee が日本市場に参入する場合、低価格 ($4/人/月) と大企業実績を武器にした攻勢が想定される。SciPOS はオールインワンの統合価値で差別化すべき

## 最近の動向（直近1年）

- **2025年3月**: SkyLake Equity Partners が BusinessOn Communication から Shiftee の75%株式を約 $280M で取得。創業者シン・スンウォン CEO が25%を保持し単独 CEO として経営を継続。BusinessOn からの独立体制へ移行
- **2025年4月**: 韓国雇用労働部と連携し、零細企業への Shiftee 無料提供プログラムを開始。社会的責任と市場浸透の両立を図る
- **2025年**: 営業利益50億ウォン（前年比3倍）を達成との報道。営業利益率75%を記録し、高収益 SaaS モデルとしての地位を確立
- **2024〜2025年**: 台湾市場でのサービス開始後2ヶ月で有料顧客400社を獲得。年内1,000社を目標とし、続いてシンガポール・日本市場への展開を計画中
- **2025年**: 「2025年社員年次有給休暇利用トレンドレポート」「2025年勤務時間分析レポート」等のデータレポートを公開し、HR 業界のソートリーダーシップを強化

## 補足: 日本版 SHIFTEE（shifteeapp.jp）について

韓国 Shiftee (shiftee.io) とは**別プロダクト**として、日本では株式会社システムサポート（石川県金沢市）が「SHIFTEE（シフティ）」（shifteeapp.jp）というシフト管理システムを提供している。月額250円/ユーザー（税抜）、25,000ユーザー以上の導入実績。飲食・小売・コールセンター等対応で AI 自動シフト作成機能を搭載。韓国 Shiftee との資本関係・提携関係は確認されていない。名称の類似に注意が必要。
