## 料金体系

Mapal OSは**カスタム見積制**を採用しており、公式サイトに定価は掲載されていない。

| 項目 | 内容 |
|------|------|
| 初期費用 | 非公開（導入コンサルティング含む見積） |
| 月額料金 | 非公開。事業規模・導入モジュール・拠点数に応じたカスタム見積 |
| 課金単位 | 拠点数・従業員数・導入するSuiteの組み合わせによるカスタム |
| 無料プラン | なし |
| 無料トライアル | デモリクエスト制（セルフサービスのトライアルは確認できず） |
| ハードウェア費用 | 非該当（SaaS専業。ただし打刻端末用にBYODタブレット等が必要） |

補足: エンタープライズ向けのため、個人店・小規模店舗にとっては導入ハードルが高い可能性がある。Capterra/GetApp上でも具体的な価格情報は未掲載。

### 差別化機能
- **Productivity Optimisation Engine**: 過去の売上パターンから最適なシフトスケジュールをAIが数クリックで生成。収益最大化・コスト最小化・サービス品質維持を同時に実現
- **買収による統合プラットフォーム**: Flow（LMS）、GetCompliant（コンプライアンス）、Easilys（食品原価管理）、Guudjob（エンゲージメント）、Inpulse（在庫・仕入AI）を統合し、ホスピタリティの「バックオフィスOS」を構築
- **CashRec**: POS売上とキャッシュの自動照合。シフト終了時の精算効率化

## 強み / 弱み（SciometaPOS との比較視点）

### 強み
1. **ホスピタリティ経営者による創業**: Lurueña家がスペイン第2位のBurger Kingフランチャイズを運営した経験から生まれたプロダクト。現場の課題を深く理解した設計
2. **エンタープライズ級の規模**: 45,000〜50,000+拠点、54カ国展開。KFC・Burger King・Starbucks・Hilton等のグローバルブランドが導入。スケーラビリティが証明済み
3. **買収による包括的プラットフォーム**: 6社以上の買収を通じてLMS・コンプライアンス・在庫・エンゲージメントを統合。「ホスピタリティのバックオフィスOS」としてのポジショニング
4. **強力なPEバッキング**: Eurazeo（仏、Mid-large Buyout）が過半数株主。PSGも少数株主として残留。M&A・欧州展開のための資金力が潤沢
5. **AI活用が進んでいる**: 売上予測→シフト最適化、AIコンテンツ生成（LMS）、Inpulse AI在庫予測など、複数領域でAIを実装済み

### 弱み
1. **POSを持たない**: 自社POS機能がなく、LastPOS等との連携に依存。POSとシフト管理の統合体験ではSciometaPOSに劣る
2. **セルフオーダー・KDS機能なし**: フロントオフィス（注文・接客）領域は完全にスコープ外。飲食店の全体最適化には別途システムが必要
3. **中小規模店舗に不向き**: 価格非公開・デモ制のため、個人店・小規模チェーンにとって導入障壁が高い。SkelloやComboのようなSMBフレンドリーさに欠ける
4. **買収統合のリスク**: 6社以上の異なるプロダクトを統合しており、UXの一貫性やデータ統合に課題がある可能性
5. **POS連携が限定的**: 連携先のLastPOSはニッチなプレイヤー。主要POS（Lightspeed、Toast、Square等）との連携エコシステムが弱い

### SciometaPOSとの競合関係
- **直接競合: 低〜中**（Mapal OSはバックオフィス統合プラットフォームであり、POSやセルフオーダー機能は持たない。シフト管理領域では競合するが、ターゲット規模が異なる）
- **間接競合**: SciometaPOSが欧州のチェーン市場に進出した際、シフト管理・勤怠管理領域でMapal Workforce Suiteと競合する可能性
- **差別化ポイント**: SciometaPOSはPOS＋セルフオーダー＋シフト管理の統合ソリューション。Mapal OSはバックオフィス特化。SciometaPOSの強みは「フロント〜バック一気通貫」であり、Mapalが手薄なフロントオフィスをカバーしている点

## 最近の動向（直近1年）

1. **Eurazeoが過半数株式を取得（2025年2〜9月）**: 2025年2月に発表、9月に取引完了。Eurazeoの Mid-large Buyoutチームによる投資。PSG Equityは少数株主として残留。欧州ホスピタリティSaaS市場のリーダーとしてのポジショニングを強化
2. **Inpulse買収（2025年12月）**: フランスのAI在庫・仕入管理プロバイダーInpulseを買収。3,500+レストラン・ベーカリー・フードキオスクが利用。Eurazeo体制下での初のM&A。バックオフィスプラットフォームの「在庫・調達・フードコスト」領域を補完
3. **50,000+拠点・54カ国に拡大**: 導入拠点数が継続的に拡大。特に北欧（Espresso House 550店舗、7-Eleven 500+店舗）での導入実績が拡大
4. **売上$20.5M〜$35M**: 186名のチームで$20.5M売上（GetLatka報告）。別ソースでは$35Mとも。PE主導の成長フェーズ
5. **AI機能の強化**: Mapal Studio（AI学習コンテンツ生成）、Workforce AIスケジューリング、Inpulse AI在庫予測など、複数プロダクトラインでAI機能を継続強化中
