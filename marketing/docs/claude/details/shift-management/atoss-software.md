## 機能比較

| 機能 | 対応状況 | 備考 |
|------|----------|------|
| POS レジ機能 | **非対応** | WFM専業、POS機能なし |
| セルフオーダー（キオスク / テーブル / QR） | **非対応** | 注文系機能は範囲外 |
| KDS（キッチンディスプレイ） | **非対応** | 飲食オペレーション機能なし |
| 在庫管理 | **非対応** | WFMに特化 |
| シフト・勤怠管理 | **充実** | コア機能。AI需要予測、自動シフト作成、勤怠管理、残業管理、コンプライアンスチェック |
| 予約管理 | **非対応** | 顧客予約機能なし |
| CRM / ロイヤルティ | **非対応** | 従業員管理に特化 |
| デリバリー連携 | **非対応** | |
| 会計ソフト連携 | **限定的** | SAP連携あり（SAPパートナー）。その他ERPとの連携可能 |
| レポート・分析 | **充実** | 人件費分析、労働生産性レポート、カスタマイズ可能なダッシュボード |
| 多言語対応 | **対応** | 多言語サポートあり |
| オフライン対応 | **不明** | クラウドベースが主力、オフライン対応の情報なし |
| AI / 自動化機能 | **充実** | AI需要予測（最大85%精度向上）、自動スケジューリング（品質最大90%向上）、非生産時間最大30%削減 |

### 主要WFM機能の詳細

- **勤怠管理（Time & Attendance）**: 自動打刻データ収集、残業・休暇管理、労働法コンプライアンス自動チェック
- **シフト作成（Workforce Scheduling）**: 需要予測ベースの最適化シフト作成、従業員の希望・スキル考慮
- **需要予測（Demand Forecasting）**: 過去データ・販売トレンド・注文パターン・イベント情報を活用したAI予測
- **従業員セルフサービス**: モバイルアプリ（ATOSS Staff Center）で休暇申請、シフト確認・交換、勤務希望提出、Face ID / Touch ID対応
- **人員コスト計画**: 戦略的なキャパシティ・需要スケジュール管理

## 強み / 弱み（SciPOS との比較視点）

### ATOSS の強み

1. **35年以上の実績**: 1987年創業、エンタープライズWFM市場で圧倒的なブランド力と信頼性
2. **AI需要予測の成熟度**: 長年のデータ蓄積とAI投資により、高精度な需要予測・自動スケジューリングを実現
3. **コンプライアンス対応**: 欧州各国の複雑な労働法規制に対する深い知見と自動コンプライアンスチェック
4. **財務的安定性**: 上場企業、高利益率（EBIT 34-36%）、潤沢な現金残高（1.6億ユーロ超）
5. **SAP連携**: SAPパートナーとして大企業のHRエコシステムにシームレスに統合
6. **Gartner認定**: 2026年Gartner Market Guide for Retail WFM Technologyで代表ベンダーに選出
7. **大企業顧客基盤**: Lufthansa、Deutsche Bahn、ALDI等の大手企業との実績

### ATOSS の弱み（SciPOS にとっての機会）

1. **飲食業特化ではない**: 汎用WFMであり、レストラン固有のニーズ（ピーク時間帯管理、チップ配分等）への最適化が弱い
2. **POS・注文管理機能がない**: WFM専業のため、POS・セルフオーダー・KDS等との一体化ができず、飲食店は複数システムが必要
3. **中小規模店舗には過剰**: エンタープライズ向け設計のため、個人店・小規模チェーンには価格・機能とも過剰
4. **導入コストが高い**: 初期費用（ライセンス・コンサル・導入）が高額、ROI回収に時間がかかる
5. **セルフサービス型導入不可**: デモ→営業→見積りの従来型販売プロセス、即時利用開始できない
6. **DACH偏重**: 売上の80%以上がDACH地域、アジア・北米でのプレゼンスは限定的
7. **飲食オペレーション連携なし**: 在庫管理・デリバリー連携・予約管理等、レストラン運営に必要な機能が欠如

### SciPOS の差別化ポイント

- **オールインワン**: POS + セルフオーダー + KDS + シフト管理を一体提供、飲食店は1つのシステムで完結
- **飲食業特化**: レストラン固有のワークフロー（ピーク管理、メニュー連動シフト等）に最適化
- **中小〜中堅向け価格設定**: 個人店から中規模チェーンまで手の届く価格帯
- **即時導入可能**: セルフサービス型でクイックスタート可能
- **アジア・欧州マルチリージョン**: ATOSSがカバーしきれない市場でのポジション構築が可能

## 最近の動向（直近1年）

### 2025年
- **通期業績**: 売上1億8,930万ユーロ（+11%）、EBIT 6,810万ユーロ（マージン36%）を達成し、過去最高を更新
- **クラウド移行加速**: クラウド・サブスクリプションARRが1億130万ユーロに達し、前年比+28%成長。初めてARR1億ユーロの大台を突破
- **ATOSS Software SEへの名称変更**（2024年5月に実施済み、SE化により欧州展開を加速）
- **ヘルスケア分野の拡大**: ベルリン・シャリテ病院など大型新規顧客を獲得
- **国際展開**: パリオフィス、ユトレヒトオフィスの開設、インド法人設立

### 2026年
- **Q1業績**: 売上5,140万ユーロ（+11%）、クラウド・サブスクリプション2,700万ユーロ（+27%、全売上の53%）
- **通期ガイダンス引き上げ**: EBITマージン予想を32%以上から34%以上に上方修正
- **Gartner認定**: 2026年Gartner Market Guide for Retail Workforce Management Technologyで代表ベンダーに選出
- **SAP連携強化**: SAPパートナーシップを通じた新規顧客獲得が継続、特にヘルスケア分野で成果
- **AI機能強化**: AI搭載のワークフォース最適化機能の継続的な改善・拡充
