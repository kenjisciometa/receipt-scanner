- **2020年頃**: マルタで創業。小規模・独立レストラン向け手頃な予約システムとして立ち上げ

- **2024年1月**: AIデジタルアシスタントAIMA・AIVAを正式ローンチ
- **2024年**: AIMA のInstagram対応拡張
- **2025年4月1日**: Tableo Add-On Marketplace（アドオンマーケットプレイス）を発表

## 料金体系

### ヨーロッパ向け価格（年払い時）

| プラン | 月額（年払い） | 主な機能 |
|--------|-------------|---------|
| **Freemium** | 無料 | 月100カバーまで。基本予約管理。ユーザー1名 |
| **Organise** | €49.17/月 | 無制限予約、インタラクティブフロアプラン、ユーザー3名 |
| **Grow** | €82.50/月（人気） | Organise機能全部 + 決済処理、SMS通知、イベント管理、ユーザー5名 |
| **Expand** | €140.83/月 | 全機能 + レストランチェーンクロスセル、AIメッセージングツール、詳細レポート、ユーザー無制限、専任アカウントマネージャー、電話サポート |

### 追加費用

| 項目 | 内容 |
|------|------|
| 年払い割引 | 月払い比で**17%割引** |
| 支払い処理 | **1% + Stripe手数料**（デポジット・プリペイ） |
| SMS / WhatsApp | キャリア費用別途 |
| 超過カバー | 無料プランの100カバー超過時は有料プランへのアップグレードを促す。有料プランでは$0.25/超過予約 |
| 複数店舗 | 1店舗ごとにプラン費用が発生。5店舗以上はネゴシエーション可 |
| 無料トライアル | 無料プラン自体がトライアル相当（期間制限なし） |

## 強み / 弱み（SciometaPOS との比較視点）

### 強み

1. **手頃な価格**: €49.17/月からスタート。無料プラン（月100カバー）あり。独立小規模店の参入障壁が低い
2. **シンプルで使いやすい**: 複雑な機能を排除した直感的UI。設定が容易
3. **AI自動化（AIMA/AIVA）**: Facebook・Instagram・電話の問い合わせをAIが24時間対応。小規模店でもスタッフ負担ゼロ
4. **コミッションなし**: 自チャネル予約はコミッション不要（Stripe決済手数料のみ）
5. **アドオンマーケットプレイス**: 必要な機能だけを追加する柔軟なモジュール構造
6. **政府系資金（MCST）**: Maltaの起業支援を受けており安定した開発基盤
7. **マーケットプレイス依存なし**: TheForkのようにマーケットプレイス露出に依存しないため純粋なソフトウェア提供に徹する

### 弱み

1. **機能範囲が極めて限定的**: 予約・フロアプラン・ゲストCRMのみ。POS、KDS、在庫、シフト、デリバリー、会計は全て対象外
2. **マーケットプレイスがない**: TheFork・Quandooのように消費者向け集客プラットフォームがなく、集客は店舗自身に依存
3. **規模が小さい**: VCからの大型調達なし。製品ロードマップと開発スピードに制約
4. **エンタープライズ機能が弱い**: 大規模チェーン・グループ管理には不向き
5. **分析機能が限定的**: Expandプランのみで基本的なKPI。高度な需要予測・ビジネスインテリジェンスはなし
6. **ブランド認知度が低い**: マルタ発の小規模プレーヤー。欧州全土での認知はTheFork/OpenTableに遠く及ばない
