## 強み / 弱み（SciPOSとの比較視点）

### 強み
1. **国内最大シェア** — 累計19,000店舗超。飲食店向け予約システムでNo.1の実績
2. **LINEヤフー子会社化** — LINE（国内1億人超）との深い統合が進行中。LINEレストランプラスにより予約・モバイルオーダー・CRM・POSを一体提供
3. **POS連携の幅広さ** — 10メーカー以上のPOSとAPI連携
4. **グルメサイト連携** — 20以上のグルメサイトから予約自動取り込み
5. **AI電話対応** — トレタ予約番で24時間予約受付
6. **UIの使いやすさ** — 飲食店スタッフ向けに設計された直感的UI

### 弱み
1. **POS本体は非提供** — 予約管理特化。POS機能自体はパートナー依存（→ SciPOSはPOS一体型）
2. **海外展開が限定的** — 日本市場中心（→ SciPOSは欧州展開済み）
3. **料金が非公開** — 比較検討しづらい。中小店舗には割高感
4. **LINEヤフー依存リスク** — 戦略がLINEエコシステムに集中する可能性
5. **セルフオーダー/キオスク非対応** — モバイルオーダーはLINEレストランプラス経由で今後提供予定

### SciPOSとの差別化ポイント
- SciPOSはPOS + セルフオーダー + 予約を一体提供できる点が差別化要因
- 欧州市場でのプレゼンスはトレタにない（日本特化）
- トレタは予約管理の深さ・グルメサイト連携の幅が圧倒的に強い

## 調査メモ

- LINEヤフーの子会社化は日本の飲食店DX市場における最大の変革。LINE上での予約・注文・決済・CRMが一体化する「LINEレストランプラス」は、初年度無料という攻撃的な価格戦略で市場シェア拡大を狙う
- 競合（TableCheck、ebica等）にとっては大きな脅威。特に中小飲食店市場でのLINEの圧倒的リーチは他社では対抗困難
- SciPOSにとっては、日本市場参入時の最大のライバルとなる可能性が高い。ただし、セルフオーダーステーション型の提供モデルはトレタ/LINEレストランプラスにはまだ弱い領域
