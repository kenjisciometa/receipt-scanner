## 強み / 弱み（SciPOSとの比較視点）

### 強み
1. **グローバル展開力** — 23言語・35か国対応は日本発の予約システムで最強
2. **送客手数料ゼロ** — グルメサイトの手数料モデルと対照的なポジショニング
3. **高級店・ホテル向けブランド** — ミシュラン星付き店舗やホテルグループの信頼が厚い
4. **キャンセル対策** — デポジット・事前決済機能が充実。ノーショー問題に強い
5. **25サイト一元管理** — 国内外グルメサイトからの予約を自動取り込み
6. **独立系** — 特定プラットフォームに依存しない中立ポジション

### 弱み
1. **中小店舗への訴求力が弱い** — 高級店・大規模店向けイメージが強い
2. **POS非提供** — 予約管理特化（→ SciPOSはPOS一体型）
3. **LINE連携が弱い** — LINEヤフー/トレタ連合に対して不利
4. **セルフオーダー/キオスク非対応**
5. **国内シェアではトレタに劣る**

### SciPOSとの差別化ポイント
- TableCheckの多言語対応（23言語）はSciPOSの参考になる
- 高級店向けのブランディングはSciPOSとターゲットが異なる
- SciPOSはPOS + セルフオーダーの一体型で差別化可能

## 調査メモ

- 「Dining Connected」構想 — 世界中のレストランとゲストをつなぐプラットフォームを目指す
- シリーズSの「S = Supporter」は、飲食業界のリーディングカンパニーが出資者に名を連ねる戦略的ラウンド。業界内での信頼の厚さを示す
- LINEヤフー/トレタ連合とは異なるポジショニング（グローバル・独立・高級店）を維持する方針
- SciPOSが欧州から日本市場を見る場合、TableCheckのグローバル戦略は参考事例となる
