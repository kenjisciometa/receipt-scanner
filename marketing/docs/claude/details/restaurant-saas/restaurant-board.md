## ターゲット市場

- **業態**: レストラン、カフェ、居酒屋等、幅広い飲食店
- **規模**: 個人店〜中規模店舗（特にコスト重視の小規模店に強い）
- **地域**: 日本全国
- **ポジション**: 「無料で始められる予約台帳」。Airレジ（リクルート）エコシステムの一部

## 強み / 弱み（SciPOSとの比較視点）

### 強み
1. **完全無料の基本機能** — コスト面で圧倒的。小規模店舗にとって最も導入しやすい
2. **リクルートエコシステム** — Airレジ・ホットペッパーグルメ・Airペイ等との統合
3. **ホットペッパーグルメ連携** — 日本最大級のグルメサイトからの予約を自動取り込み
4. **IVRy連携のAI電話対応** — 24時間電話予約受付（2024年10月〜）
5. **リクルートの信頼性とサポート**
6. **Airレジ連携** — POS + 予約管理の一体利用が可能

### 弱み
1. **ホットペッパーグルメ依存** — 他のグルメサイト連携は限定的（食べログ等との連携弱い）
2. **機能の深さ** — トレタやebicaと比較すると機能面で見劣り
3. **無料ゆえのサポート制限** — 有料サービスほどのサポートは期待しづらい
4. **グローバル展開なし** — 日本国内限定
5. **セルフオーダー/キオスク非対応**
6. **高級店向けの機能不足** — カジュアル店舗向け

### SciPOSとの差別化ポイント
- レストランボードの「無料」は強力だが、機能面ではSciPOSが上回る
- SciPOSはPOS + セルフオーダーの一体型で差別化
- Airレジ + レストランボードの組み合わせがSciPOSの日本市場での最も近い競合モデル

## 調査メモ

- リクルートの「Airブランド」（Airレジ、Airペイ、レストランボード等）は日本の中小飲食店で圧倒的なシェアを持つ
- 「無料」の予約台帳は市場参入障壁を大幅に下げており、有料サービスにとって脅威
- ただし、ホットペッパーグルメ以外のグルメサイト連携が弱く、「ホットペッパーグルメを使わない店舗」にとっては利点が限定的
- SciPOSが日本市場に参入する場合、「Airレジ + レストランボード」の無料エコシステムが最大の価格面でのハードルになる
