### 沿革

- **2007年**: LaFourchette（「フォーク」の意）としてパリで創業。創業者：Bertrand Jelensperger、Patrick Dalsace、Denis Fayolle
- **2011年**: フランス初のオンライン予約プラットフォームとして確立
- **2014年**: Tripadvisorが約$150Mで買収
- **2018年**: イタリア、オランダ、ポルトガル、オーストラリアなど11カ国に拡大
- **2019年**: MICHELINガイドの優先予約プラットフォームに認定
- **2020年**: 各国ブランド（LaFourchette/El Tenedor/Bookatable等）を「TheFork」に統一リブランド
- **2021年**: Almir Ambeskovic がCEOに就任
- **2023年**: 新ブランドアイデンティティ発表。「Revenue Management」ツール（データ分析）提供開始
- **2024年11月**: MastercardとEurope全土での多年間パートナーシップ締結
- **2025年2月**: OpenAI Operator（ChatGPT）の欧州唯一のローンチパートナーに選出。英国でAI予約自動化を開始
- **2025年夏**: TheFork Manager サマーアップデート（後述）

## 料金体系

TheFork Managerは公式に価格を非公開とし、問い合わせ制を採っている。

| 項目 | 内容 |
|------|------|
| 初期費用 | 非公開（要問い合わせ） |
| プラン構成 | Performance（最も人気）/ Enterprise（複数店舗グループ向け） |
| 月額料金 | 非公開（カスタム見積もり） |
| コミッション | TheFork/Tripadvisor経由の予約に対し **約€2.60/カバー**（人数課金） |
| 直接予約 | ウェブサイト・SNSからの直接予約は**コミッション無料**（サブスク内） |
| 無料プラン | なし |
| 無料トライアル | 非公開（デモ申し込み可） |
| 料金体系の特徴 | サブスク固定費 + TheForkマーケットプレイス経由のカバーコミッション（二重構造） |

## 強み / 弱み（SciometaPOS との比較視点）

### 強み

1. **欧州最大の予約マーケットプレイス**: 55,000+店舗、月間2,000万+訪問者。TheForkに掲載されるだけで集客力が得られる
2. **MICHELINガイド・Tripadvisor・Mastercardとの強固なパートナーシップ**: 信頼性とブランド力が圧倒的
3. **OpenAI Operator統合（2025年）**: 欧州唯一のChatGPTローンチパートナー。AI予約エージェントとしての地位確立
4. **グループ・エンタープライズ機能**: 複数拠点のクロスセル、集中データ管理
5. **マーケティング自動化**: Meta/Google広告のワンクリック設定、メールキャンペーン
6. **YUMSロイヤルティ**: 消費者側のリピート来店を促進
7. **多言語対応（10言語）**: 欧州多国籍展開に強い

### 弱み

1. **POS・オペレーション機能がない**: KDS、在庫、シフト管理、会計が全て外部依存。SciometaPOSのようなオールインワンには対抗不能
2. **コミッション二重構造**: サブスク料金 + カバーコミッション（約€2.60/人）がコスト圧力になりやすい
3. **直接予約チャネルへの誘導が難しい**: プラットフォーム依存が深いほどコミッション負担増
4. **価格の不透明性**: 公式サイトに価格非公開。交渉力のない小規模店には不利
5. **オーストラリア以外はアジア・北米展開なし**: 欧州特化プラットフォームの地理的制約
6. **Tripadvisor依存**: 親会社の経営戦略変化が直接影響するリスク
