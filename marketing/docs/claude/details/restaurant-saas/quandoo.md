## 基本情報

| 項目 | 内容 |
|------|------|
| 会社名 / プロダクト名 | Quandoo |
| 本社所在地 | ベルリン、ドイツ |
| 設立年 | 2012年 |
| 親会社 | Recruit Holdings（日本）。2015年3月に約€198.6M（全額現金）で買収 |
| 資金調達（買収前） | 累計$33M（4ラウンド）。主要投資家: DN Capital, Sixt, Piton Capital。最終ラウンド: Series D（2014年10月） |
| 従業員規模 | 約370名（50カ国出身） |
| 主な展開地域 | ドイツ、英国、イタリア、オーストリア、スイス、フィンランド、オランダ、トルコ、シンガポール、オーストラリア、ニュージーランド（計11カ国） |
| パートナー店舗数 | 16,000+店舗 |

### 沿革・閉鎖経緯

- **2012年**: ベルリンで創業
- **2014年10月**: Recruit Holdingsが7.09%を取得（Series D）
- **2015年3月**: Recruit Holdingsが残92.91%を約€198.6Mで取得し完全子会社化
- **2020年**: COVID-19の影響で数百万ユーロの損失計上。親会社から€30.6Mのローン調達
- **2021年**: 87名のリストラ実施
- **2026年3月**: Recruit Holdingsが「戦略的ポートフォリオレビュー」の結果として「計画的かつ秩序ある事業終了」を発表
- **2026年9月30日**: 新規予約受付終了、コンシューマー向けサービス終了
- **2026年12月31日**: 全インフラ（サイト、アプリ、サーバー）シャットダウン予定
- **閉鎖まで**: ロイヤルティポイントの獲得は6月30日まで。最終6ヶ月間のサービス手数料を免除

## 強み / 弱み（SciometaPOS との比較視点）

### 強み

1. **欧州+APACの広範なマーケットプレイス**: 16,000+店舗、11カ国に存在感
2. **Recruit Holdings傘下**: HotPepper（日本）、AirRegiとのシナジー。「世界最大のレストランマーケティング＆ソフトウェア企業」グループの一員
3. **シンプルなカバーコミッション課金**: 固定費不要で始めやすい（小規模店向け）
4. **APACでの認知**: シンガポール、オーストラリア、NZでの存在感

### 弱み

1. **2026年12月31日に完全閉鎖**: **最大の弱み。競合として評価不要、市場から退出確定**
2. **機能が予約のみに限定**: POS、在庫、KDS、シフト管理、マーケティングなし。オールインワンプラットフォームには大きく劣る
3. **TheForkと競合して南欧で苦戦**: フランス・スペイン・イタリアではTheForkに完敗
4. **2020年から赤字継続**: COVID後に回復できず、親会社からの融資に依存
5. **AI・自動化機能の欠如**: 技術的投資が不十分
6. **Recruit Holdingsの戦略的優先度低**: 採用（Indeed、Glassdoor）が本業であり、レストラン予約は周辺事業

## 競合への示唆

Quandooが閉鎖する16,000+の加盟店舗は移行先を探している。**ドイツ、英国、イタリア、オーストリア、スイスを中心に数千店舗が2026年後半に新しいプラットフォームへの切り替えを検討**。TheFork、OpenTable、aleno、Sugarvinetablesなどが積極的に移行キャンペーンを展開中。
