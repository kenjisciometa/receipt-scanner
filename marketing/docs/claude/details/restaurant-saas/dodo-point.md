## ハードウェアモデル

- **店頭iPad**: 顧客が電話番号を入力するための専用タブレット
- **消費者側**: ハードウェア不要（電話番号入力のみ）
- **管理画面**: PC/タブレットからWeb管理

## 強み / 弱み（SciPOSとの比較視点）

### 強み
1. **究極のシンプルさ** — 電話番号入力だけ。カード不要・アプリ不要・会員登録不要
2. **消費者の導入ハードル最小** — 電話番号を入力するだけで即ポイント付与
3. **低コスト** — 月額最大4,400円。飲食店の負担が極めて小さい
4. **韓国での実績** — 10,000店舗・1,200万ユーザーの大規模実績
5. **パーソナライズドマーケティング** — 購買行動データに基づく自動配信

### 弱み
1. **電話番号ベースの限界** — LINE連携やアプリベースの体験には及ばない
2. **日本市場での知名度が低い** — 韓国発サービスで日本での認知が限定的
3. **予約管理・テーブル管理は非対応** — ポイント特化
4. **POS連携が弱い**
5. **LINE連携なし** — 日本市場ではLINE連携が重要だが非対応
6. **日本の導入店舗数が不明** — 近年の日本市場での成長度が不透明
7. **プライバシー懸念** — 電話番号という個人情報をタブレットに入力することへの抵抗

### SciPOSとの差別化ポイント
- dodo pointの「電話番号だけ」アプローチはSciPOSのロイヤリティ設計の参考になる
- ただし、日本市場ではLINE連携の方が受け入れられやすい
- SciPOSはPOS + セルフオーダー + ロイヤリティを一体提供できる点で大きく差別化

## 調査メモ

- dodo pointは「世界最もシンプルなポイントシステム」を標榜。電話番号入力のみという極限のシンプルさが売り
- 韓国では大きな成功を収めたが、日本市場ではLINE連携やアプリベースのソリューション（トレタスタンプ、レストランスター等）との競争が激しい
- 日本での最新情報が限定的であり、日本市場での成長が鈍化している可能性がある
- SciPOSにとって直接的な脅威ではないが、「シンプルさ」「低コスト」のアプローチは参考になる
- Max Brenner、Croissant Chou Zaku Zaku等のチェーン導入実績あり
