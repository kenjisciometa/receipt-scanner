## ターゲット市場

### 対応業態

- 独立系レストラン・カフェ・バー（メインターゲット）
- ビストロ・ベーカリー・フードトラック
- ホスピタリティ全般（独立店〜小規模チェーン）

### 対象規模

- **メインターゲット**: 独立オーナー・個人店〜小規模チェーン（SMB）
- 「ご近所のレストランや飲食店」を明示的なターゲットとして訴求
- 大規模エンタープライズチェーンは対象外

### 主要地域

- ドイツ（最大市場）
- オーストリア・スイス（DACH圏）
- フランス（展開中）
- Nexiグループとしてヨーロッパ各国へのISV向け決済プラットフォーム展開

## ハードウェアモデル

- **orderbird PRO**: iPad専用（BYOD対応。自社iPad使用可）
- **orderbird MINI**: 独自オールインワン端末（TSE認証済み。SIM内蔵、レシートプリンタ内蔵、カードリーダー内蔵、バーコードスキャナー内蔵）
- **決済端末**: Nexi統合カードリーダー（スターターパッケージに含む）
- **プリンタ**: キッチンプリンタ/バープリンタ対応（複数ロケーションへの印刷振り分け）
- **ハンドヘルド**: iPhone/iPod touchでのワイヤレス注文

## 強み / 弱み（SciometaPOS との比較視点）

### 強み

1. **DACH市場のNo.1クラウドPOS**: ドイツで最もよく知られるiPad POSシステム。17,000+顧客の信頼性
2. **ドイツ法規制完全準拠**: TSE（技術的セキュリティ設備）認証。ドイツ・オーストリア・スイス・フランスの税務法規制対応
3. **DATEV会計連携**: ドイツの主要会計ソフトDATEVとのネイティブ連携は独自優位
4. **Nexiグループの決済ネットワーク**: ヨーロッパ最大級の決済企業Nexiとの統合で決済インフラが強固
5. **使いやすさ**: SMB向けに特化した直感的UI。スタッフ教育コストが低い
6. **orderbird MINI**: オールインワンの移動可能端末でフードトラック等にも対応

### 弱み

1. **シフト管理の欠如**: 専用のシフト・勤怠管理機能なし。SciometaPOSの統合シフト管理は明確な差別化
2. **地域集中**: DACH+フランスに限定。日本・アジア市場への展開実績ゼロ
3. **KDS機能の限定性**: 本格的なKDS専用スクリーン対応が不明確
4. **エンタープライズ対応の限界**: 大規模チェーン・フランチャイズ管理は不向き
5. **料金の不透明さ**: 公式料金ページが頻繁にアクセス制限。実際の料金体系が把握しにくい
6. **Nexi統合後の独立性**: Nexiの大企業文化との統合によるスタートアップらしい機動性の低下懸念

## 最近の動向（直近1〜2年）

- **2024年5月**: NeximとorderbirがドイツのISV向け「Nexi Partner Portal（NPP）」を発表。48時間以内での決済統合を実現する次世代決済プラットフォームとして展開
- **2024年**: orderbird PROのQRコード注文システム強化。4-in-1（オンライン注文 + キオスク + テーブルQR + 電話注文）への進化
- **2026年1月**: ドイツの消費税（MwSt）変更対応アップデート
- **Nexiグループ戦略**: SMB向けに「完全な決済+POS統合」として位置づけ。ヨーロッパ各国への展開をISVパートナー経由で加速
- **競合動向**: Gastrofix（Lightspeed傘下）やVectron（Shift4傘下）との競争が続く中、Nexiの決済ネットワーク優位を活かした差別化を推進
