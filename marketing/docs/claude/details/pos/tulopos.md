# TuloPOS (RavinteliSofta Oy)

## Company Overview
- **Legal Name:** RavinteliSofta Oy
- **Y-tunnus:** 2636488-1
- **Website:** https://www.tulopos.fi
- **HQ Location:** Helsinki, Finland
- **Founded:** 2014
- **CEO:** Juhani Iisakki Ollila-Herlevi
- **Founders:** Juhani Ollila and Veli-Matti Ojala (restaurant industry professionals)
- **Employees:** 7 (as of December 2023)
- **Revenue:** EUR 1.65M (2023); profit EUR 125K
- **Ownership:** Private, founder-owned

## Product
- **Core Product:** TuloPOS - cloud-connected POS system designed by restaurant professionals for restaurants
- **Key Features:**
  - POS register with customizable display layouts
  - Kitchen order management (printer-based)
  - Inventory and stock management
  - Split payment and bill splitting
  - Chain management (shared/separate products, tiered user permissions)
  - Real-time dashboard with remote access
  - Event/party portable checkout stations
  - Electronic receipts
  - Digital ID verification
  - Electronic delivery notes (waybills)
  - Online store module (Nettikauppa)
  - Accounting integration (automatic daily/monthly data transfer)
  - Invoice generation (one-click)
  - Quick error correction
  - Hourly cloud backups with local operation capability
- **Supported Platforms:** Proprietary software on POS terminals. Cloud dashboard accessible from any browser
- **Integrations:**
  - **Payment:** Card payment terminal integration (provider not specified)
  - **Accounting:** Procountor, Fennoa (direct integration)
  - **Benefits:** Epassi (lunch benefit QR code)
  - **Hardware:** Receipt printers, matrix printers, card payment devices, networking equipment
