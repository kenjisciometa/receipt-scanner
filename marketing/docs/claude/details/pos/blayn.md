## 強み / 弱み（SciPOSとの比較視点）

### 強み

1. **券売機一体型**: POS + 券売機 + セルフレジを一社で提供。券売機が必要な飲食店に最適
2. **飲食店完全特化**: 飲食店の現場ニーズに徹底的に最適化
3. **プリンター内蔵端末**: 省スペースのオールインワン設計
4. **4ヶ国語対応**: 日英中韓の多言語メニュー＆音声案内。インバウンド対応
5. **無人店舗対応**: セルフレジ+券売機で完全無人化が可能
6. **低コスト月額**: 月額¥0〜¥3,080から始められる
7. **iPhoneハンディ**: 手持ちのiPhoneを注文端末として最大7台利用可能
8. **クラウド多店舗管理**: 本部から一括でメニュー変更・価格配信

### 弱み

1. **飲食以外非対応**: 小売・サービス業には利用不可
2. **券売機が高額**: ¥759,000〜の初期投資が必要
3. **専用端末依存**: iPad/Android汎用端末非対応（ハンディ以外）
4. **国際展開なし**: 日本国内のみ
5. **エコシステムが限定的**: 連携サービスが少ない
6. **KDS非対応**: 専用キッチンディスプレイなし
7. **シフト管理なし**: 勤怠管理機能がない
8. **デリバリー連携なし**: フードデリバリーとの統合なし
9. **企業規模が小さい**: サポート・開発リソースに制約
10. **情報の透明性**: 料金や企業情報の公開が限定的

### SciPOSとの差別化ポイント

- blaynの券売機機能は、SciPOSのセルフオーダーキオスクと直接比較可能
- ただしblaynは日本の飲食店（ラーメン、定食等）向けの券売機。SciPOSはレストラン向けセルフオーダー端末
- blaynの4ヶ国語対応は、SciPOSの多言語対応と比較対象
- SciPOSはPOS + キオスク + QR + デリバリー + シフト + KDSの統合プラットフォーム。blaynはPOS + 券売機のシンプルな構成
- blaynの無人店舗対応は、日本市場の人手不足トレンドに対応した独自の強み

## 調査メモ

- 券売機市場での独自ポジション。ラーメン店・定食屋などの券売機需要に特化
- SNSマーケティング事業から飲食店POS/券売機へ転換した異色の経歴
- 六本木ヒルズにオフィスを構え、青森にも拠点。開発は青森の可能性
- 7,000店舗契約はニッチ市場では健闘。ただしAirレジやスマレジとは桁が異なる
- 券売機の価格（¥759,000〜）は高額だが、人件費削減効果を訴求
- 無人店舗ソリューションは、日本の深刻な人手不足に対応するトレンドとして注目
- 4ヶ国語対応は、インバウンド需要の多い観光地の飲食店に訴求
