## 機能比較

### Trivec（POS部門）の機能

| 機能カテゴリ | 対応状況 | 備考 |
|-------------|---------|------|
| **POS レジ機能** | ✅ | テーブル管理、注文管理、分割払い、フロアプラン |
| **セルフオーダー（QR）** | ✅ | QRコードスキャンによるオーダー・決済（アカウント不要、アプリ不要） |
| **セルフオーダー（キオスク）** | ✅ | セルフサービスキオスク（カード決済対応） |
| **モバイルオーダー・決済** | ✅ | HandyPay（テーブルサイドでのオーダー＋決済を統合したハンドヘルド端末） |
| **KDS（キッチンディスプレイ）** | ✅ | Trivec Display（キッチン/バー/パストリー等のワークステーション別に表示） |
| **在庫管理** | ✅ | 連携対応（Apicbase等） |
| **シフト・勤怠管理** | 連携のみ | Planday 等の外部スタッフ管理システムと連携 |
| **予約管理** | ✅ | 予約・テーブルブッキング機能あり |
| **CRM / ロイヤルティ** | ✅（部分） | ギフトカード、ロイヤルティ機能 |
| **デリバリー連携** | ✅ | デリバリー管理機能あり |
| **会計ソフト連携** | ✅ | Pennylane（フランス向け）等の会計ソフト連携 |
| **レポート・分析** | ✅ | MyTrivecウェブポータル（クラウドダッシュボード、リアルタイム売上データ） |
| **多言語対応** | ✅ | スウェーデン語/英語/フランス語/オランダ語等（北欧+ベネルクス+フランス展開） |
| **オフライン対応** | ✅ | 不明（安定性が強みとして言及） |
| **飲料管理システム** | ✅（独自） | Beverage Control System（バー/ホテル向け飲料管理・自動化）—Trivec独自の強み |

### Caspeco（管理プラットフォーム）の機能

| 機能カテゴリ | 対応状況 | 備考 |
|-------------|---------|------|
| **POS** | ✅ | クラウドPOS |
| **スタッフ管理** | ✅ | スケジューリング、タイムトラッキング、給与、売上レポート |
| **予約管理** | ✅ | ゲスト向け・スタッフ向け両対応 |
| **分析** | ✅ | パフォーマンストラッキング、計画ツール |
| **オンライン注文** | ✅ | 対応あり |

### 統合後のフルスタック
合算すると: **POS + スタッフ管理 + 予約 + 分析 + QR注文 + キオスク + KDS + 飲料管理** を1プラットフォームで提供

## 強み / 弱み（SciometaPOS との比較視点）

### 強み
- **北欧POSのデファクトスタンダード**（スウェーデン・ノルウェー・デンマークでの圧倒的実績）
- **飲料管理システム（Beverage Control）**: バー・ホテルに特化した飲料量管理は競合が持ちにくい独自機能
- **HandyPay**: オーダーと決済を一体化したハンドヘルドでスタッフ効率化
- **6,500+顧客の大規模プラットフォーム**: ネットワーク効果と安定した収益基盤
- **Trivec（1998年〜）の長い歴史と信頼性**
- **Caspeco統合によるフルスタック化**: POS + スタッフ管理 + 予約 + 分析の1プラットフォーム
- **SEK 256M（約€23M）の安定した収益規模**

### 弱み
- **シフト管理はCaspeco側の機能**でTrivec単体では外部連携のみ（統合後は解消）
- **価格非公開**でエントリーバリアが不明
- **北欧・西欧に特化**しており日本・アジア展開実績なし
- **統合途上のリスク**: Caspeco-Trivec統合は進行中で機能・サポートの均一化に時間がかかる可能性
- **Windowsとウェブベース**の端末が主体でモバイルファーストではない部分も（HandyPayで対応中）

### SciometaPOS との差分
| 比較軸 | Trivec/Caspeco | SciometaPOS |
|--------|---------------|-------------|
| シフト管理 | Caspeco統合後は対応（Trivec単体は外部連携） | 統合済み |
| 飲料管理 | 独自の飲料制御システム（強み） | 不明 |
| セルフオーダー | QR + キオスク + HandyPay | 対応あり |
| 対象市場 | 北欧 + 西欧 | 日本/アジア |
| 顧客規模 | 6,500+ | 不明 |
| 主要業態 | バー・ホテルに特に強い | レストラン全般 |

## 最近の動向（直近1〜2年）

- **2023年1月**: Monterro（北欧B2B PE）がCaspecoへのマジョリティ投資を実施。
- **2024年〜**: Trivecがパートナーエコシステム強化戦略を公式発表。外部連携でのプロダクト拡充を加速。
- **2025年6月**: Caspeco が Verdane から Trivec を買収。ディールアドバイザーはClipperton（Paris）。
  - 合算6,500+顧客、欧州最大級レストランテックプラットフォームへ。
  - Trivecはフランス・ベルギー・オランダの拠点をCaspecoグループへ持ち込み。
- **2026年4月**: フランス向けスタッフシステムおよびベルギー向けテーブル予約の機能ロールアウトを予定・実施。
- **統合ブランド**: 全サービスを「Caspeco」ブランドで統一、Trivecは「Trivec by Caspeco」として継続展開。
