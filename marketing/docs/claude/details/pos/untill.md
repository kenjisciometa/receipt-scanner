## 料金体系

### unTill Air（クラウドPOS・小規模向け）

| プラン | 月額（年間契約） | 主な機能 |
|--------|----------------|---------|
| Basic | €21.85/月 | 注文管理、ダイレクト販売モード、ビジネスインサイト、デジタル請求 |
| Standard | €33.85/月 | Basic＋請求書・プロフォーマ印刷、キッチン/バーチケット印刷 |
| Hospitality Extra | €44.85/月 | Standard＋テーブルプラン、コース管理、メニュー作成 |

- 年間割引率: 約15%（月払いは割増）
- 無料トライアル: **14日間（クレカ不要）**

- **カスタム料金**（規模・機能・リセラー経由で変動）
- 125+インテグレーション対応

### ハードウェア費用

- BYOD対応（iOS / Android タブレット・スマートフォン）
- 専用端末・決済端末（AMS1、P400 Plus、V400m、L2s Pro 等）
- unTill Payments: トランザクションベース料金（公開なし）

## 強み / 弱み（SciometaPOS との比較視点）

### 強み

- **30年以上の実績**: オランダ・ベルギーで最も認知度の高いホスピタリティPOSの一つ
- **二層構造**: 小規模向け Air（低価格・14日トライアル）と大規模向け Prime（フル機能）で幅広い顧客をカバー
- **125+統合**: 予約、会計、デリバリー、スタッフ管理など豊富な連携先
- **多国展開実績**: ベネルクス中心に西欧・北欧7カ国以上でリセラーネットワーク確立
- **AI Magic Menu**: AI機能の実装で差別化を図りつつある

### 弱み

- **シフト・勤怠管理が自社不在**: 統合スタッフ管理は外部連携のみ
- **セルフオーダーが外部依存**: キオスク・QRオーダーは Jamezz 等パートナーに依存
- **カスタマーサポート評価**: 一部レビューで初期費用・解約時の対応に不満
- **アジア展開なし**: 日本語対応・アジア市場の実績なし
- **SciometaPOS との重複**: POS + 連携型統合 vs SciometaPOS の完全統合（POS+セルフオーダー+シフト管理）
