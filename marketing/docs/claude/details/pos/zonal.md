## 料金体系

### ソフトウェア料金

- **カスタム価格制**: 規模・要件に応じた個別見積もり
- **レンタルパッケージ（公開最低価格）**: £229/月〜（EPoS + ハンドヘルドPOSセット）
  - 初期前払金: £1,100（導入サービス込み）
  - 最低契約期間: 36ヶ月
- **月額目安**: £70〜£120+（ハードウェア別途、小規模サイト）
- **3 for 1パッケージ**: EPoS + オンライン注文 + 予約を一括提供（詳細価格は要問合せ）

### 初期費用

- 導入費用（プロジェクトマネジメント、設定、トレーニング）は見積もり次第
- 大規模チェーン向けは完全カスタム価格

### 無料プラン / トライアル

- 無料プランなし
- トライアル: デモのみ（詳細は要問合せ）

### ハードウェア費用

- ハンドヘルドPOS（iServe）: レンタルパッケージに含む場合あり

## 強み / 弱み（SciometaPOS との比較視点）

### 強み

1. **英国ホスピタリティの圧倒的実績**: 45年以上の歴史。Wetherspoon・Greene King等の最大手が採用。「英国で最も信頼されるEPoS」と評される
2. **統合エコシステム**: EPoS + 予約 + KDS + 在庫 + CRM + ロイヤルティ + PMS をネイティブ統合。200+外部統合
3. **シフト・スタッフ管理内蔵**: ローテーション作成・給与監視機能が標準搭載
4. **大規模チェーン対応**: 16,000+拠点の管理。マルチサイト集中管理に優れる
5. **受賞実績**: Restaurant's Readers' Choice Awards「Best EPoS 2024」受賞

### 弱み

1. **高コスト・長期契約**: £229/月〜、36ヶ月最低契約。小規模独立店には重すぎる
2. **柔軟性の低さ**: カスタム専用ハードウェアが前提。BYOD不可
3. **導入の複雑さ**: 大企業向け設計のため、中小規模では過剰スペック
4. **アジア市場対応なし**: 日本語・中国語等への対応や、アジア税務規制への対応ゼロ
5. **移行期の不確実性**: 2026年2月のVolaris買収後、製品ロードマップが不明確
6. **価格非透明性**: 見積もり対応のみで、比較検討しにくい

## 最近の動向（直近1〜2年）

- **2026年2月**: Volaris Group（CSI/Constellation Software傘下）がZonalを買収。Jesper Ulstedが暫定CEOに就任。Volarisは「buy-and-keep」方針で長期保有を表明
- **2025年**: Dojo（ペイメントフィンテック）とのパートナーシップ強化。テーブルサイドオーダー&決済パッケージ展開
- **2024年**: Restaurant's Readers' Choice Awards「Best EPoS 2024」受賞
- **2024年**: R&D投資が£6.9Mに増加（40周年記念の成長投資）
- **2022年**: Airship（CRM・ロイヤルティプラットフォーム）を買収し、マーケティング機能を大幅強化
- **買収後の方向性**: Volaris傘下での「次世代ソフトウェア開発加速」を宣言。独立性維持を強調
