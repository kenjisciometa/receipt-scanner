## 強み / 弱み（SciPOSとの比較視点）

### 強み

1. **国内ターミナルPOSシェア1位（約36〜40%）**: 圧倒的な市場支配力
2. **豊富なハードウェアラインナップ**: POS本体から自動釣銭機、セルフレジ、タブレットまで全て自社製品
3. **東芝グループの信頼性**: 連結売上5,770億円の大企業グループ
4. **セルフレジ・セルフオーダーの豊富なソリューション**: WILLPOS-Self、Relax Order NS等
5. **長い歴史と実績**: 1950年設立、70年以上のPOS事業経験
6. **グローバル展開**: 日本だけでなく海外にもPOS事業を展開
7. **エンタープライズ向けの堅牢性**: ターミナルPOSの安定性・耐久性
8. **省人化の実績**: セルフオーダー・セルフレジ導入で1店舗あたり1日3〜4時間の勤務時間削減

### 弱み

1. **非常に高価**: 1台150万〜300万円はSMB市場では受け入れられない価格帯
2. **導入のハードルが高い**: 見積もり→販売代理店経由→工事→保守契約と複雑
3. **クラウド/SaaS化が遅い**: 基本はオンプレミス型ターミナルPOS
4. **CRM・ロイヤルティが弱い**: 顧客管理機能は基本レベル
5. **モバイルファーストではない**: iPadPOS（POSasy）はあるがメイン製品ではない
6. **スタートアップ的な柔軟性がない**: 製品アップデートの頻度・スピードが遅い
7. **多言語対応が不明確**: 具体的な対応言語情報が公式サイトで明示されていない
8. **LINE連携等のモダンCRMがない**: dinii等の新興勢力と比較してデジタルマーケティング機能が弱い

## 調査メモ

- 東芝テックは日本のPOS市場で「旧来型の王者」。ターミナルPOSでは圧倒的なシェアを持つが、タブレットPOS・クラウドPOS市場ではAirレジ、Square、スマレジ等の新興勢力に押されている
- 飲食店向けではFScompass NSが主力だが、価格帯が高く（150万〜300万円）、SMBには不向き
- セルフレジ（WILLPOS-Self）、セルフオーダー（Relax Order NS）、スマホオーダー（OtegaruOrder）と幅広いソリューションを持つが、統合性はdinii等の一気通貫型には劣る
- SciometaPOSとの直接競合度は低い: 東芝テックは日本のエンタープライズ市場、SciometaPOSはSMB〜中規模市場。ただし中規模チェーン市場では競合の可能性あり
- 東芝テックのグローバル展開はSciometaPOSの将来的な参考になる
- POSasy（iPad対応）は東芝テックのクラウド化への橋渡し製品だが、まだメイン製品ではない
