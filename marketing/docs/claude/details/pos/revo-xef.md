## 料金体系

### プラン構成
Revo XEF は **One / Basic / Plus / Pro** の4プランを提供。ただし価格は公開されておらず、問い合わせ・デモ申込み形式。

| プラン | 特徴 | 価格 |
|--------|------|------|
| One | エントリー向け（衛星アプリ非対応） | 非公開 |
| Basic | 基本衛星アプリ対応 | 非公開 |
| Plus | 上位機能を含む中間プラン | 非公開 |
| Pro | フルスイート対応 | 非公開 |

- **課金単位**: 店舗 / 端末ベース（SaaS月額）
- **無料トライアル**: あり（App Store経由で確認）
- **ハードウェア費用**: 別途（Revo HANDY端末、Revo STAND等）

## 強み / 弱み（SciometaPOS との比較視点）

### 強み
- スペイン国内でタブレット型POSのマーケットリーダー
- 20カ国対応・10言語対応でスペイン語圏LatAmへの展開実績
- Cegidグループ傘下による財務安定性・エンタープライズ対応力
- 50+統合エコシステム（デリバリー、予約、PMS、会計をフルカバー）
- ホテルチェーン（Meliá等）への大型導入実績
- モジュール型設計でSMBからエンタープライズまで対応

### 弱み
- **シフト・勤怠管理は自社非搭載**（外部連携のみ）
- 料金非公開でエントリーバリアが不明
- iOS（iPad）専用のため Android BYOD不可
- AI・自動化機能はGeniusモジュール等に依存し成熟度は不明
- 日本語・アジア言語対応の記載なし

### SciometaPOS との差分
| 比較軸 | Revo XEF | SciometaPOS |
|--------|----------|-------------|
| シフト管理 | 外部連携のみ | 統合済み |
| セルフオーダー | 追加コスト | 標準 |
| 対応OS | iOS専用 | クロスプラットフォーム |
| 対象市場 | スペイン/LatAm/欧州 | 日本/アジア |
| ホテル連携 | 強み（PMS多数） | 未確認 |
