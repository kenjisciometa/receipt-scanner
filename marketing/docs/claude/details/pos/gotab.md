## 強み / 弱み（SciometaPOS との比較視点）

### 強み
- **コンタクトレス注文のパイオニア**: QRオーダー+サーバー注文のハイブリッドモデルは独自性が高い
- **フードホール・マルチベンダー対応**: 単一QRで複数ベンダー統合は他社にない機能
- **デバイスアグノスティック**: Windows/iOS/Android完全BYOD。HW初期投資最小化
- **透明な料金体系**: $15〜$229/月の段階的プラン。アドオンも明示
- **ブルワリー・タップルーム特化**: 最初のビール2分以内提供の実績
- **急成長**: 2025年 YoY 150%成長。$500M+ GMV/年
- **柔軟な契約**: 長期契約不要。0%ファイナンス対応
- **Opsi買収**: 在庫・レシピ・タスク管理機能を内製化

### 弱み
- **小規模チーム**: 約49名。エンタープライズサポートの限界
- **北米中心**: 39州+カナダ。グローバル展開は未着手
- **多言語非対応**: アジア市場には不向き
- **シフト管理弱い**: 基本的な権限管理のみ。本格的なシフトスケジューリングは外部依存
- **決済手数料やや高め**: 2.40% + $0.15は業界標準より高い
- **アドオン積み上げ**: 必要な機能を全て追加すると月額が大幅に増加
- **エンタープライズ実績限定**: 大規模チェーン向けの実績はPARやToastと比較して不足

### SciometaPOS との比較
- GoTabのQRオーダー+サーバーハイブリッドモデルはSciometaPOSのセルフオーダーアプローチと類似
- SciometaPOSはシフト管理を自社統合 → GoTabは弱い
- SciometaPOSはアジア市場・多言語対応 → GoTabは北米英語のみ
- GoTabのフードホール・マルチベンダー機能はSciometaPOSにはないユースケース
- GoTabの透明な段階的料金モデルはSciometaPOSの価格設計の参考になる
- GoTabのデバイスアグノスティックアプローチはSciometaPOSと同様のBYOD哲学
- 双方向メッセージング（キッチン↔ゲスト）は革新的な参考機能
