## 強み / 弱み（SciometaPOS との比較視点）

### 強み
- **予約管理の強さ**: 他のAndroid POS競合に比べてOnline Reservationへの統合が深い。コミッション0%での予約受付はコスト意識の高い独立系飲食店に訴求力がある
- **圧倒的な低価格**: €34.90/月（縛りなし）は市場最安値クラス。追加デバイスも€19.90/月と低廉
- **ノーコミットメント**: 月単位での解約自由が明確に打ち出されている
- **Android対応**: iPadが高価な市場でAndroid汎用機対応はコスト優位
- **NF525認証**: フランス法規制対応済み
- **オフライン対応**: 通信不安定環境でも使用可能
- **包括的機能（200+）**: 小規模事業者向けにすべての機能を低価格で提供
- **1ヶ月無料トライアル**: 試用コスト0で導入検討可能

### 弱み
- **規模感の小ささ**: SARL、資本金€20,000の小規模企業。エンタープライズ対応は難しい
- **キオスク（セルフオーダー端末）**: 対応確認されず
- **AI機能**: なし
- **デリバリープラットフォーム連携**: Uber Eats等との直接API連携は不明
- **サポート品質**: レビューサイトで顧客サポートの対応速度に関する不満あり
- **iPad非対応**: Apple製品ユーザーへの訴求ができない
- **国際展開**: フランス国内のみ
- **エンタープライズ機能**: フランチャイズ管理、大規模チェーン向け機能は弱い
- **2023年の値上げ**: 16.7%の値上げはSMB向けとしては注意すべき点

### SciometaPOSとの差別化ポイント
- ConnectillはSMBとコスト重視の独立系飲食店が主な対象。SciometaPOSが対象とするアジア・日本市場とは競合しない
- SciometaPOSの**統合型シフト管理**はConnectillより大幅に高機能
- ConnectillのGoogleやSNSとの**予約統合の深さ**は参考になる設計思想
- Connectillの€34.90/月というシンプル低価格モデルはSMB向け価格設定の参考事例
