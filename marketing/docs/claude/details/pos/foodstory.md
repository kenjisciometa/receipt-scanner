## LINE MAN Wongnai について

- タイ最大のフードデリバリー・ライフスタイルプラットフォーム
- 2023年に100,000の新規レストランをオープン
- IPOを検討中（2025年にタイまたは米国で上場予定との報道）
- FoodStory買収により、POS市場でのNo.1ポジションを確保

### 追加サービス
- **FoodStory Market**: レストラン向けオンライン食材・原材料マーケットプレイス

## ハードウェアモデル

- iPad/タブレットベース（iOS対応）
- 専用ハードウェアパッケージあり（POS + プリンター + 周辺機器）
- 自社製造ハードウェアは持たない

## 強み/弱み（SciPOSとの比較視点）

### 強み
- LINE MAN Wongnai傘下によるデリバリー連携の優位性
- タイ国内No.1 POS市場リーダー
- 11年以上の業界経験
- 200以上の機能を搭載した包括的プラットフォーム
- 在庫管理のリアルタイム追跡が充実
- FoodStory Market（食材マーケットプレイス）による独自のバリューチェーン
- 手頃な価格設定（年間10,800～32,000 THB）
- CRM・ロイヤリティ機能が充実

### 弱み
- タイ国内限定
- 多言語対応が限定的（タイ語/英語のみ）
- LINE MAN Wongnai傘下のため、独立した海外展開の自由度が低い可能性
- 日本語対応なし
- グローバルブランドチェーンへの導入実績は限定的

## 最近の動向

- **2023年7月**: LINE MAN Wongnaiに買収される
- **2023年**: LINE MAN Wongnaiが100,000の新規レストラン開拓、POS市場No.1を目標
- **2024年5月**: LINE MAN WongnaiがIPOを検討中（タイまたは米国、2025年目標）
- **2024年10月**: KBIZ向け特別プロモーション（Wongnai POS + FoodStory POS）

## 調査メモ

- FoodStoryはLINE MAN Wongnai買収により、「デリバリー + POS」の統合エコシステムの一部となった
- タイPOS市場のリーダーとして、SciPOSがタイ市場に参入する際の最大の競合
- FoodStory Market（食材マーケットプレイス）は、POS企業がバリューチェーンを拡大するユニークな事例
- 価格帯はSciPOSと比較可能な範囲。タイ市場ではSciPOSの「多言語対応」「日本食特化」が差別化ポイントになりうる
- LINE MAN WongnaiのIPO成功時には、さらなる成長投資が予想される
