### ポジショニング
SkyTabはShift4の**次世代フラッグシップPOS**。Harbortouch、Revel、Future POS、POSitouchなどの旧ブランドを統合し、Shift4のPOS戦略の中核を担う。2026年5月に「Shift4 Dine」として最終リブランド予定。

## 料金体系

| 項目 | 詳細 |
|------|------|
| **初期費用** | **無料**（ハードウェア無償提供） |
| **月額料金** | $29.99/端末/月 |
| **含まれるもの** | ハードウェア、ソフトウェア、設置、24/7サポート、生涯保証、オンラインオーダー、ロイヤルティ、予約・ウェイトリスト |
| **契約期間** | 30日間トライアル → 36ヶ月契約（早期解約ペナルティなし） |
| **決済手数料（標準）** | 2.75% + $0.15/取引（フラットレート） |
| **決済手数料（大口）** | 月商$50K+で2.4〜2.75% + $0.10〜$0.15に交渉可能 |
| **Advantage Program** | 0%実効（カード顧客に3〜4%サーチャージ転嫁） |
| **年間手数料** | Program Fee $250 + Regulatory Assurance Fee $189.99〜$325（合計 ~$400〜$500/年/端末） |
| **無料プラン** | なし |
| **トライアル** | 30日間（契約前に繁忙期テスト可能） |

### HarbortouchとSkyTabの違い
実質的にHarbortouchはSkyTabに統合済み。料金・機能は同一。SkyTabが現行ブランド。

## 強み（SciometaPOS比較視点）

1. **圧倒的コストパフォーマンス** — $29.99/月でHW＋SW＋サポート＋オンラインオーダー＋ロイヤルティ＋予約がすべて込み。北米市場で最もコスパの高いオールインワンPOS
2. **ゼロ初期投資** — ハードウェア無償提供により参入障壁が極めて低い
3. **使いやすさ** — 「1シフトでスタッフが習熟」と評価されるUI
4. **キオスク＋QRオーダー** — 2024年秋以降、セルフオーダー機能が充実
5. **Shift4の決済インフラ** — 大規模プロセッサの信頼性
6. **24/7米国拠点サポート** — 常時対応
7. **Advantage Program** — 処理手数料ゼロ化オプション
8. **Bar Rescue提携** — Jon Taffer氏によるブランド認知

## 弱み（SciometaPOS比較視点）

1. **決済プロセッサ縛り** — Shift4専用。日本・アジア市場では利用不可。SciometaPOSのローカル決済対応が優位
2. **英語のみ** — 多言語非対応。SciometaPOSの多言語UIが明確な差別化
3. **アジア展開なし** — 日本・アジア市場への対応予定なし
4. **隠れ年間手数料** — $400〜500/年/端末が公開料金に含まれず不透明
5. **36ヶ月契約** — 長期コミットメント要求（ただし早期解約ペナルティなし）
6. **シフト管理が弱い** — ネイティブのスケジューリング機能なし。SciometaPOSの統合シフト管理が強み
7. **在庫管理が基本的** — 詳細な食材レベル管理は未対応
8. **BYOD不可** — 専用HWのみ。既存機器の流用不可
9. **メニュー修飾子の設定が煩雑** — 複雑なメニュー構成での使い勝手に課題

## 最近の動向（直近1年）

- **2024年秋**: SkyTab Kiosk（セルフオーダーキオスク）リリース。バンプバー対応追加
- **2024年秋**: QR PayをMulti-Round QR Orderingにアップグレード
- **2024年6月**: Shift4がRevel Systems（$250M）とVectron（€85M）を同時買収。SkyTabへの機能統合を計画
- **2025年**: Shift4が決済手数料値上げ・年間手数料の複数回請求で加盟店から不満
- **2025年2月**: Shift4がGlobal Blueを$1.5Bで買収
- **2026年5月**: SkyTab → **Shift4 Dine**にリブランド予定（機能・価格変更なし）
- **プロモーション**: $5,000サインアップクレジット実施中（2026年4月、16枠中8枠使用済み）
- **2026年4月**: Maple AIとShift4が提携し、AI電話注文機能を導入
