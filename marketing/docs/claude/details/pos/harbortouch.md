## 料金体系

| 項目 | 詳細 |
|------|------|
| **初期費用** | **無料**（ハードウェア無償提供、決済契約付き） |
| **月額料金** | $29.99/端末/月 |
| **契約期間** | 3年間（早期解約金あり、自動更新） |
| **決済手数料** | 標準: 2.75% + $0.15/取引 |
| **Advantage Program** | 0%実効（顧客にクレジットカード3〜4%サーチャージ転嫁） |
| **年間手数料** | Program Fee $250 + Regulatory Assurance Fee $189.99〜$325（合計 ~$500/年/端末） |
| **無料プラン** | なし（HWは無料だが月額・決済契約必須） |
| **無料トライアル** | 30日間試用期間あり |

## 強み（SciometaPOS比較視点）

1. **無料ハードウェア** — 初期投資ゼロは小規模店舗にとって大きな魅力。SciometaPOSがBYODモデルで対抗可能
2. **業界最安クラスの月額** — $29.99/端末は北米市場で最安水準
3. **オールインワンバンドル** — POS＋決済＋オンラインオーダー＋ロイヤルティが月額に含まれる
4. **Shift4の決済インフラ** — 大規模決済プロセッサの信頼性とスケーラビリティ
5. **生涯HW保証** — 機器故障リスクなし
6. **Bar Rescue提携** — TV露出によるブランド認知度

## 弱み（SciometaPOS比較視点）

1. **3年契約ロックイン** — SciometaPOSが柔軟な契約で差別化可能
2. **決済プロセッサ縛り** — Shift4のみ。日本・アジアでは使えない
3. **英語のみ** — SciometaPOSの多言語対応が明確な優位性
4. **隠れコスト** — 年間手数料$500+/端末が事後的に発生
5. **アジア展開なし** — SciometaPOSの日本・アジア特化が差別化ポイント
6. **在庫管理が基本的** — SciometaPOSの詳細在庫管理が優位
7. **シフト管理が弱い** — SciometaPOSのネイティブシフト管理が明確な強み
8. **カスタマイズ性低** — 専用HW縛りでBYOD不可

- **2024〜2025年**: HarbortouchブランドをSkyTabに統合・リブランド完了。リテール・サロン・スパプランを廃止し、レストラン・バー専業化
