## 1. SciPOSの現在地と課題の要約

### 強み（守り抜くべき）
- **ビュッフェモード**: 業界唯一。最大の差別化要素
- **NFC注文**: 極少数（me&u/sundayのみ）。QRより高速なUX
- **無料コアPOS + BYOD**: 導入障壁ゼロ。Toast $0やLoyverseと同等のフリーミアム
- **統合型（POS+キオスク+KDS+シフト+在庫）**: allO（独）のみ類似
- **GPSジオフェンス打刻**: シフト専業（Deputy/7shifts）以外では稀少
- **12+言語**: 多国籍レストランや観光地で強力な武器

### 課題（対処すべき）
- **デリバリー連携なし**: 競合の80%が対応済み。市場参入の前提条件
- **ロイヤルティ/CRM未実装**: 80%の競合が提供。顧客の粘着性に直結
- **AI機能なし**: 67%の競合がAI搭載。2027年には「AI非搭載=レガシー」の認識に
- **オンライン注文なし**: デリバリー手数料回避の直接注文ニーズが急増
- **会計連携がフィンランド限定**: 海外展開の前提条件

## 3. Go-to-Market戦略

### 3.1 ターゲットセグメント優先順位

```
第1ターゲット ──→ 第2ターゲット ──→ 第3ターゲット
   ↓                  ↓                  ↓
食べ放題/ビュッフェ   多国籍レストラン     独立系SMB全般
（差別化最大）        （多言語で差別化）    （無料コアで獲得）
```

#### セグメント1: 食べ放題・ビュッフェレストラン
| 項目 | 詳細 |
|------|------|
| **なぜ最優先か** | ビュッフェモードは業界唯一。Jamezzのラウンド管理のみ類似だがPOS非保有 |
| **市場規模** | 北欧・DACH・日本に多数。アジアンビュッフェ、寿司食べ放題、ホテルビュッフェ等 |
| **訴求メッセージ** | 「食べ放題専用モードのある唯一のPOS」 |
| **営業アプローチ** | 食べ放題店舗リスト作成 → デモ → 2週間無料トライアル |
| **LTV期待値** | 高（ビュッフェモードがスイッチングコストを形成） |

#### セグメント2: 多国籍・観光地レストラン
| 項目 | 詳細 |
|------|------|
| **なぜ第2か** | 12+言語対応は競合でSyrve（20+）/Loyverse（30+）程度しかなく稀少 |
| **市場規模** | 観光都市（ヘルシンキ、コペンハーゲン、ベルリン、ロンドン等）に集中 |
| **訴求メッセージ** | 「メニューもスタッフUIも12言語。観光客にもスタッフにも」 |
| **営業アプローチ** | 観光協会/レストラン組合への提携 → レファラルプログラム |

#### セグメント3: 独立系SMB（新規開業〜小規模チェーン）
| 項目 | 詳細 |
|------|------|
| **なぜ第3か** | 無料コアPOS + BYODは最も低い導入障壁。市場最大のセグメント |
| **競合の対応** | Toast $0/Square 無料/Loyverse 無料 → フリーミアム戦争の最前線 |
| **訴求メッセージ** | 「お手持ちのiPad/タブレットで今すぐ無料スタート。POS+キオスク+KDS+シフト管理まで」 |
| **営業アプローチ** | SEO/コンテンツマーケティング → セルフサービスオンボーディング |

## 6. 競合別対抗戦略

### 6.1 最重要競合への個別対策

#### vs Tebi（最高脅威）
| Tebiの強み | SciPOSの対抗策 |
|-----------|---------------|
| 成功報酬型（月商€10Kごとに€20） | 無料コア+定額制の予測可能性を訴求。「売上に応じて費用が増えない安心感」 |
| €56M調達、CapitalG支援 | 機能差別化で勝負（ビュッフェ/NFC/シフト管理はTebi未対応） |
| 無制限デバイス | STANDARDプランで端末無制限を提供 |
| 英国展開開始 | 北欧/DACHでのプレゼンスを先に確立 |

#### vs Trivec/Caspeco（スウェーデン最大競合）
| Trivecの強み | SciPOSの対抗策 |
|-------------|---------------|
| 8,000+顧客、スウェーデン市場支配 | 無料コアでの新規開業獲得。Trivecは高額なためコスト意識の高い店舗を狙う |
| POS+予約+スタッフ+分析の統合 | キオスク/NFC/ビュッフェの差別化。Trivecのキオスク機能は弱い |
| 2025年のCaspeco統合で強化 | AI機能（売上予測）で次世代感を訴求 |

#### vs Restolution（フィンランド最大競合）
| Restolutionの強み | SciPOSの対抗策 |
|------------------|---------------|
| 4,000+店舗、40年の実績 | 無料スタートでスイッチングリスクをゼロに。「まず無料で試してから決めてください」 |
| 北欧3カ国展開済み | ビュッフェモード + NFC + AI（Phase 2以降）で差別化 |
| €14M売上の安定企業 | 統合型（シフト管理/在庫管理込み）vs Restolutionの別ツール運用 |

#### vs allO（ドイツ最類似競合）
| allOの強み | SciPOSの対抗策 |
|-----------|---------------|
| SciPOSに最も類似する機能構成 | NFC/ビュッフェ/GPSシフト/12+言語で差別化 |
| €25〜/月の低価格 | 無料コアで更に低い導入障壁。機能面での優位性を訴求 |
| $6.5M調達、1,000+拠点 | AI機能の早期投入で「次世代POS」のポジション |
