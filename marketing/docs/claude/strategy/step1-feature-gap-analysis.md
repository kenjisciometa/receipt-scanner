## 2. 競合が広く提供していてSciPOSにない機能（ギャップ）

### 🔴 クリティカルギャップ（競合の67%以上が提供）

#### G1. デリバリープラットフォーム連携
| 項目 | 詳細 |
|------|------|
| **競合普及率** | ~80%（Toast/Syrve/Flipdish/Checkmate/Olo/Owner.com等） |
| **具体的な連携先** | Uber Eats, DoorDash, Deliveroo, Wolt, Just Eat, Glovo |
| **業界リーダー** | Checkmate（30+プラットフォーム）、Deliverect（70,000+拠点） |
| **SciPOSへの影響** | **必須**。デリバリー非対応は現代のPOS市場で「自動的失格」になりつつある |
| **実装の方向性** | Deliverect/HubRise等のミドルウェア連携 or 主要3-5プラットフォーム直接連携 |

#### G2. AI機能（需要予測・メニュー最適化・アシスタント）
| 項目 | 詳細 |
|------|------|
| **競合普及率** | ~67%（何らかのAI機能を提供） |
| **主要AI機能** | 売上/需要予測（Nory 95-99%精度）、AIメニュー最適化（Bite 20%客単価↑）、AI管理アシスタント（Owner.com AI CMO/CFO）、自然言語分析（CrunchTime） |
| **業界リーダー** | Syrve（AI予測+請求書OCR+配送ルーティング）、SpotOn Profit Assist（4.3%コスト削減）、Bite Lift（天候等15+外部シグナル活用） |
| **SciPOSへの影響** | **高**。AI非搭載POSは競争力を急速に失いつつある |
| **実装の方向性** | Phase 1: AI売上予測/需要予測 → Phase 2: AIメニュー推奨/アップセル → Phase 3: AIアシスタント |

#### G3. ロイヤルティ/CRM
| 項目 | 詳細 |
|------|------|
| **競合普及率** | ~80%（ポイント/リワード/顧客DB） |
| **主要機能** | ポイント/スタンプ、リピーター特典、顧客プロファイル（アレルギー/来店履歴等）、セグメント別キャンペーン |
| **業界リーダー** | PAR Punchh（多段階リワード）、SevenRooms（包括的ゲストCRM）、Blackbird（ブロックチェーンロイヤルティ） |
| **SciPOSへの影響** | **高**。顧客リテンション機能がないと長期的な店舗粘着性が弱い |
| **実装の方向性** | デジタルスタンプ/ポイント → 顧客DB → セグメント別プロモーション |

#### G4. オンライン注文（テイクアウト/自社EC）
| 項目 | 詳細 |
|------|------|
| **競合普及率** | ~73% |
| **主要機能** | 自社ブランドWeb注文ページ、テイクアウト事前注文、コミッション無料の直接注文 |
| **業界リーダー** | Owner.com（10,000+店舗、AI最適化）、Flipdish（ホワイトラベル）、Choice（30,000+、コミッション無料） |
| **SciPOSへの影響** | **高**。デリバリー手数料30%回避のために自社注文を求める店舗が急増 |
| **実装の方向性** | シンプルなオンライン注文ページ生成 → テイクアウト管理 → ブランドアプリ |

## 5. 推奨優先順位

### 🥇 最優先（今すぐ着手）

| 順位 | 機能 | 理由 | 工数目安 |
|------|------|------|---------|
| 1 | **デリバリー連携** | 市場参入の前提条件。非対応=失格 | 中〜大 |
| 2 | **ロイヤルティ/CRM基盤** | 顧客粘着性の根幹。80%の競合が提供 | 中 |
| 3 | **オンライン注文ページ** | デリバリー手数料回避ニーズが急増 | 中 |

### 🥈 次優先（6ヶ月以内）

| 順位 | 機能 | 理由 | 工数目安 |
|------|------|------|---------|
| 4 | **AI売上予測** | 競合差別化の最前線。Nory/SpotOn等が先行 | 大 |
| 5 | **会計ソフト連携拡充** | 市場進出先に合わせて必須 | 小〜中 |
| 6 | **マーケティング（プッシュ通知）** | ロイヤルティと連動で効果倍増 | 小 |

### 🥉 中期（12ヶ月以内）

| 順位 | 機能 | 理由 | 工数目安 |
|------|------|------|---------|
| 7 | **AIメニュー最適化/アップセル** | Bite実績:客単価20%↑。ROI明確 | 中 |
| 8 | **食品原価管理（レシピレベル）** | 差別化機会。Restaurant365が$1.4B評価で証明 | 大 |
| 9 | **AI音声注文パートナー連携** | 自社開発より連携が現実的 | 小 |

## 6. 守るべき優位性

以下のSciPOSユニーク機能は、競合が真似しにくい or まだ追随していない領域。**積極的にマーケティングで訴求**すべき。

| 機能 | 訴求ポイント | ターゲット |
|------|-------------|-----------|

## 7. 競合トレンドまとめ

| トレンド | 説明 | SciPOSへの示唆 |
|----------|------|---------------|
| **AI爆発** | 67%の競合がAI機能搭載。売上予測・音声注文・自動マーケティング | AI非搭載は2027年に致命的ハンデ |

| **フリーミアム標準化** | Toast $0、Square 無料、Loyverse 無料のSMB獲得戦略 | SciPOSの無料コアは時代に合致。維持すべき |

| **規制対応の差別化** | HACCP・労働法・GDPR対応が付加価値に | EU市場ではコンプライアンス機能が武器になる |
